## 使用方式

### 本地同步（Obsidian）

1. 安装 **Obsidian Git** 插件
2. 克隆仓库：`git clone https://github.com/XINRAN168/XR-Knowledge-Base.git`
3. 用Obsidian打开该目录
4. 配置自动同步（建议5分钟间隔）

### 快速收集

1. 在 `📥收件箱/待处理/` 新建笔记
2. 写入临时想法/灵感/待整理信息
3. Git推送后，云澜自动读取并分类

### 信息流转

```
📥收件箱/待处理/ → 中转站/日收集/ → 周整理 → 八大库/
```
