## 🏃 核心流转闭环

```
📊情报·机会 → 💡灵感想法 → 🔮须弥推演 → 🚀OPC → 📝经验沉淀
      ↓                                      ↑
  💀失败归档 ←──────────────────────── (失败时)
```

### 流转五阶段

| 阶段 | 入口 | 出口 | 时限 |
|:---:|:---|:---|:---:|
| 情报收集 | 📊 | 💡 | 3天 |
| 灵感分析 | 💡 | 🔮 | 7天 |
| 推演验证 | 🔮 | 🚀/💀 | 5天 |
| 执行落地 | 🚀 | 📝 | 按计划 |
| 经验归档 | 📝 | 归档 | 3天 |

## 🚀 快速开始

### 添加新话题

1. 选择对应话题目录
2. 创建新文件（参考模板）
3. 添加对应标签

### 触发流转

直接说明意图，例如：
- "这个情报值得深入分析" → 进入灵感想法
- "需要验证这个方案" → 进入须弥推演
- "方案通过，开始执行" → 进入OPC

### 查看历史

按时间/状态/话题筛选目录即可
