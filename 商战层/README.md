## 🧠 档案标准格式

1. **核心思维** - 商业决策逻辑和思考方式
2. **底层逻辑思维** - 形成核心思维的根源
3. **核心行为准则** - 贯穿始终的行事风格
4. **核心价值观** - 商业理念和价值取向
5. **心力/情绪韧性** - 抗压能力和情绪管理
