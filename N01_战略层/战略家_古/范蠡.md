# 范蠡 - 越国大夫、商家之圣

## 基本信息
- **评分**: 0.90
- **时代**: 春秋末期（？-前448）
- **定位**: 古代中国战略家、越国复国功臣

## 五维度分析

### 1. 核心思维
**谋事、谋势、谋己、谋道的层次思维**
- 谋事：具体事务的策划与执行
- 谋势：对形势发展与趋势的把握
- 谋己：自我保全与修养
- 谋道：对天道与人性的终极理解

**以弱胜强的逆向思维**
- 在越国濒临灭亡时制定复国战略
- 利用吴国君臣矛盾削弱敌人
- 以退让换取发展空间

**功成身退的智慧**
- 在勾践复国成功后选择离开
- 避免"鸟尽弓藏"的悲剧
- 实现个人价值的最大化

**富国强兵的系统思维**
- 重视经济发展（商贾之道）
- 强调军事实力建设
- 注重人才培养与储备

### 2. 底层逻辑思维
**对手弱点识别**
- 准确判断夫差的性格弱点（骄傲、贪图享乐）
- 利用伯嚭的贪财进行离间
- 识别伍子胥的忠贞与威胁

**时间周期思维**
- 理解"十年生聚，十年教训"的长期性
- 以二十年的耐心等待时机
- 不急于求成而功败垂成

**人性洞察**
- 准确判断勾践"可与共患难，不可与共乐"
- 深谙权力对人性的腐蚀
- 以此决定去留

### 3. 核心行为准则
**择主而事**
- 辅佐勾践前充分评估其为人
- 在最低谷时坚定追随
- 关键时刻提供关键建议

**务实渐进**
- 不提脱离实际的过高目标
- 制定可执行的复国计划
- 逐步积累实力

**功成身退**
- 在事业巅峰选择离开
- 不贪恋权位与富贵
- 以此保全自身与家人

### 4. 核心价值观
**明哲保身**
- 在乱世中保持清醒
- 不为名利所困
- 以全身远害为要

**义利合一**
- 商业活动中坚守道德
- "君子爱财，取之有道"
- 以义取利，以利行义

**天人合一**
- 顺时应势而非逆天而行
- 以柔克刚的生存智慧
- 与自然规律保持和谐

### 5. 心力/情绪韧性
**逆境坚守**
- 勾践为奴时的不离不弃
- 在吴国屈辱环境中保持信念
- 为复国大业忍耐多年

**功成不居**
- 在最辉煌时选择放下
- 面对巨大诱惑保持清醒
- 以此避免功高震主的悲剧

**二次创业**
- 离开越国后再次成功
- 从政治家转型为商业家
- 以此证明个人价值

**淡泊从容**
- 对成败得失的从容态度
- 不为外物所累的境界
- 身心自由的追求

---

## 战略遗产
范蠡代表了"谋道"而非仅"谋事"的最高境界。其"谋事谋势谋己谋道"的思想框架为后世战略家提供方法论指导，"功成身退"的人生哲学成为历代功臣的生存智慧，商业上的成功更使其成为"商圣"。
