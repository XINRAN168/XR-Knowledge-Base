## 📖 使用指南

### 学习路径
1. **入门**：先读 [人性底层逻辑] 和 [中国社会底层逻辑]，理解人与社会
2. **进阶**：学习 [商业底层逻辑] 和 [金融股票底层逻辑]，理解财富运行
3. **实战**：结合 [职场管理底层逻辑]，将认知转化为行动

### 核心原则
- 每篇文档都从 **本质定义** 出发
- 重点掌握 **3-5条核心规律**
- 用 **实战案例** 验证理论
- 最终形成 **一针见血** 的洞察力
