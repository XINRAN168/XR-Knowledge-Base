**人物评分：0.88**

## 一、核心思维

### 1.1 核心理念
- **捭阖之道**：开合张弛，阴阳变化
- **反应之术**：从对方反应获取信息
- **内揵之道**：说服与进谏的技巧

### 1.2 思维特点
- **辩证思维**：阴阳相随，正反相成
- **权变思维**：因时因地因人制宜
- **信息思维**：重视信息收集与分析

### 1.3 认知框架
```
捭（开）→ 阖（闭）→ 阴阳 → 变化 → 捭阖
  ↓        ↓
显学      隐学
```

## 二、底层逻辑思维

### 2.1 纵横体系
- **捭阖之术**：
  - 捭：开启，言谈，先发制人
  - 阖：闭合，沉默，后发制人
  - 阴阳结合，灵活运用

### 2.2 预判逻辑
- **揣情摩意**：揣测对方心理
- **量权揣势**：分析形势力量
- **飞箝之术**：先奖励后控制

### 2.3 说服模型
```
量权（分析形势）→ 揣情（了解对方）→ 摩意（试探反应）→ 制心（控制情绪）→ 成事
```
