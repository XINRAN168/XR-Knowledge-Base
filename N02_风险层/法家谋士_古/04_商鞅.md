# 商鞅 - 变法先驱

## 人物画像
- **评分**: 0.85
- **时代**: 战国时期(约公元前395年-公元前338年)
- **身份**: 秦国左庶长、法家代表人物
- **核心标签**: 变法、严刑峻法、农战强国

---

## 一、核心思维

### 1.1 人性论：好利恶害
商鞅的人性假设：
- "民之性，饥而求食，劳而求佚，苦则索乐，辱则求荣"
- 趋利避害是人的本能
- 可以利用这一本性进行治国

### 1.2 法治思维
- 法律是治国的根本工具
- "壹赏，壹刑，壹教"——统一赏罚标准
- "法不阿贵"——法律面前人人平等

### 1.3 耕战思想
- 国家强大必须依靠农业和军事
- 一切资源集中于耕战
- 抑制商业和游食之民

### 1.4 变革思维
- "治世不一道，便国不法古"
- 反对因循守旧
- 根据现实需要制定政策

## 二、底层逻辑思维

### 2.1 激励机制设计
- 用赏赐引导百姓从事耕战
- 用刑罚威慑百姓遵守法令
- 赏罚分明，言行必果

### 2.2 利益对齐
- 使个人利益与国家目标一致
- 耕战有赏，其他活动无益
- "家有余粮则国富"

### 2.3 权威强化
- "权者，君之所独制也"
- 君主必须掌握绝对权力
- 以权力保障法令执行

### 2.4 效果导向
- 以实际效果评判政策
- 不问动机，只看结果
- "中程者赏，弗中程者诛"

## 三、核心行为准则

### 3.1 变法原则
| 阶段 | 内容 |
|:---|:---|
| 立威信 | 先树立法令权威 |
| 试推行 | 小范围试点验证 |
| 全推开 | 逐步推广扩大 |
| 严执行 | 违法必究 |

### 3.2 行事风格
- **雷厉风行**：令行禁止，不拖泥带水
- **铁面无私**：不畏权贵，一视同仁
- **坚韧不拔**：面对阻力不退缩

### 3.3 奖惩体系
- 奖励：军功授爵、农桑有赏
- 惩罚：私斗有罪、惰农受罚

## 四、核心价值观

### 4.1 价值序列
```
国家富强 > 法治秩序 > 农战效率 > 个人权利
```

### 4.2 核心信条
1. **以法治国**：法律是治国的唯一标准
2. **富国强兵**：国家强大是最高目标
3. **壹赏壹刑**：统一标准，公正执行
4. **不法常可**：根据实际需要变法

### 4.3 政治理想
- 建立中央集权的强大国家
- 实现富国强兵的目标
- 以法治代替人治
