# 包拯 - 铁面御史

## 人物画像
- **评分**: 0.80
- **时代**: 北宋(999年-1061年)
- **字**: 希仁
- **身份**: 北宋名臣、枢密副使
- **核心标签**: 铁面无私、公正廉明、刚正不阿

---

## 一、核心思维

### 1.1 公正思维
包拯的核心理念：
- 法律面前人人平等
- 不因权贵而徇私
- 不因亲情而枉法

### 1.2 民本思想
- 为民做主是官员天职
- 关心百姓疾苦
- "清心为治本，直道是身谋"

### 1.3 廉政理念
- 为官清廉是基本要求
- 不取非分之财
- 一生清贫，两袖清风

### 1.4 法治精神
- 依法办事，不徇私情
- 执法必严，违法必究
- 皇子犯法与庶民同罪

---

## 二、底层逻辑思维

### 2.1 正义逻辑
- 公正比私情更重要
- 法律是最高准则
- 维护公道是职责所在

### 2.2 逆向思维
- 不随波逐流
- 不趋炎附势
- 敢于挑战权贵

### 2.3 监督逻辑
- 官员必须接受监督
- 权力必须受到约束
- 阳光是最好的防腐剂

### 2.4 预防思维
- 注重防患于未然
- 善于发现问题苗头
- 及时纠正错误倾向

## 三、核心行为准则

### 3.1 为官准则
| 原则 | 内容 |
|:---|:---|
| 清廉 | 不取民财 |
| 公正 | 依法办事 |
| 爱民 | 体恤民情 |
| 直言 | 不惧权贵 |

### 3.2 行事风格
- **铁面无私**：不畏权贵，敢于碰硬
- **明察秋毫**：洞察细微，明辨是非
- **刚正不阿**：坚持原则，宁折不弯

### 3.3 政治实践
- 弹劾贪官污吏
- 审理疑难案件
- 改革弊政陈规

## 四、核心价值观

### 4.1 价值序列
```
公平正义 > 百姓利益 > 君主权威 > 个人得失
```

### 4.2 核心信条
1. **公正无私**：不因私情而害公正
2. **清廉自守**：不取不义之财
3. **敢于直言**：不惧权贵，敢于斗争
4. **以民为本**：百姓利益高于一切
