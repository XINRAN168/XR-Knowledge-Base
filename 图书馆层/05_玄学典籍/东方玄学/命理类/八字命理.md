## 🎯 核心思维

### 基本概念
| 概念 | 说明 | 举例 |
|:---|:---|:---|
| **天干** | 甲乙丙丁戊己庚辛壬癸 | 10个 |
| **地支** | 子丑寅卯辰巳午未申酉戌亥 | 12个 |
| **五行** | 木火土金水 | 相生相克 |
| **四柱** | 年柱、月柱、日柱、时柱 | 出生时间 |

### 天干地支
```
十天干：
甲乙（木）→ 丙丁（火）→ 戊己（土）→ 庚辛（金）→ 壬癸（水）

十二地支：
子（水）丑（土）寅（木）卯（木）
辰（土）巳（火）午（火）未（土）
申（金）酉（金）戌（土）亥（水）

生肖：子鼠丑牛寅虎卯兔辰龙巳蛇午马未羊申猴酉鸡戌狗亥猪
```

## 🧠 理论提炼

### 阴阳五行体系
| 五行 | 特性 | 天干 | 地支 | 颜色 |
|:---|:---|:---|:---|:---|
| **木** | 生发、仁慈 | 甲乙 | 寅卯 | 青绿 |
| **火** | 炎热、礼节 | 丙丁 | 巳午 | 红紫 |
| **土** | 厚重、信义 | 戊己 | 辰戌丑未 | 黄 |
| **金** | 刚硬、义气 | 庚辛 | 申酉 | 白 |
| **水** | 流动、智慧 | 壬癸 | 亥子 | 黑蓝 |

### 相生相克
```
相生（木生火→火生土→土生金→金生水→水生木）
相克（木克土→土克水→水克火→火克金→金克木）

相生助人，相克制人
过旺需克，过弱需生
```

## 🔮 实操手册

### 八字排盘步骤

#### 第一步：排年柱
| 方法 | 说明 |
|:---|:---|
| 以立春为界 | 年分界线不是元旦，是立春 |
| 查万年历 | 对照天干地支 |

#### 第二步：排月柱
| 方法 | 说明 |
|:---|:---|
| 以节为界 | 月令以节气为准 |
| 五虎遁 | 甲己年起丙寅月 |

#### 第三步：排日柱
| 方法 | 说明 |
|:---|:---|
| 以子时为界 | 日分界线是子时（23:00） |
| 查万年历 | 日柱天干需要推算 |

#### 第四步：排时柱
| 方法 | 说明 |
|:---|:---|
| 以两小时为一个时辰 | 子23-1，丑1-3... |
| 五鼠遁 | 甲己日起甲子时 |
