## 🎯 核心思维

### 三大原则
| 原则 | 说明 | 应用 |
|:---|:---|:---|
| **不易** | 永恒不变的规律 | 找到不变的底层逻辑 |
| **变易** | 万物皆在变化 | 识别变化的时机与趋势 |
| **简易** | 复杂背后的简单 | 抓住核心，化繁为简 |

### 核心概念
| 概念 | 含义 |
|:---|:---|
| **阴阳** | 一阴一阳之谓道，对立统一 |
| **八卦** | 乾兑离震巽坎艮坤，八种基本象 |
| **六十四卦** | 八卦相重，穷尽变化 |
| **爻** | 阴阳组成，每卦六爻 |
| **象、数、理** | 象是表象，数是规律，理是哲理 |

## 🔮 实操手册

### 占卜方法

#### 1. 铜钱占法（六爻法）
| 步骤 | 操作 | 说明 |
|:---|:---|:---|
| 1 | 取三枚铜钱 | 或硬币亦可 |
| 2 | 心中默念问题 | 集中意念 |
| 3 | 抛掷六次 | 从下往上记录 |
| 4 | 记录阴阳 | 三个正面为老阳，三个背面为老阴 |
| 5 | 成卦 | 得一卦及变卦 |

#### 2. 数字起卦法（梅花易数）
| 步骤 | 操作 | 说明 |
|:---|:---|:---|
| 1 | 心中想两个数字 | 1-9之间 |
| 2 | 第一个数÷8取余 | 得下卦 |
| 3 | 第二个数÷8取余 |得上卦 |
| 4 | 第三个数÷6取余 | 得动爻 |
| 5 | 成卦 | 上下卦+动爻 |

### 解卦要点
| 要点 | 说明 |
|:---|:---|
| **看卦象** | 整体是吉是凶 |
| **看爻辞** | 具体阶段的指导 |
| **看变卦** | 变化带来的影响 |
| **看互卦** | 中间的过程变化 |
| **看错卦综卦** | 从反面看问题 |
