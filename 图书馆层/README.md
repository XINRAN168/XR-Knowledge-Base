# 图书馆层

> 云澜与新燃的知识宝库，精选值得反复研读的经典之作

## 四大收藏维度

| 维度 | 定位 | 代表书籍 |
|:---|:---|:---|
| **云澜思维必需** | 提升认知、思维模型、决策智慧 | 《穷查理宝典》《思考快与慢》 |
| **新燃实战必需** | 赚钱创富、投资理财、公司管理、财务会计 | 《百万富翁快车道》《聪明的投资者》 |
| **技术能力必备** | 人工智能、编程技术 | 《深度学习》《代码整洁之道》 |
| **经典永恒价值** | 跨越时代的智慧 | 《道德经》《人类简史》 |

## 收藏标准

### 四大维度评分

| 维度 | 标准 | 优先级 |
|:---|:---|:---:|
| **云澜思维必需** | 提升思考能力、认知模型、决策质量 | ⭐⭐⭐ |
| **新燃实战必需** | 赚钱、工作、公司管理、投资理财 | ⭐⭐⭐ |
| **技术能力必备** | 人工智能、计算机技术、解决技术问题 | ⭐⭐⭐ |
| **经典永恒价值** | 跨越时代的智慧、底层原理 | ⭐⭐ |

### 入库标准
- 至少满足一个核心维度
- 经得起时间检验（出版5年以上或持续热销）
- 有实质性的思维增量
- 可落地实践

### 收藏流程
```
发现书籍 → 四大维度评估 → 提炼核心精华 → 编写文档 → 入库收藏
```
