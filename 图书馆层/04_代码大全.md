# 《代码大全》 Code Complete
> 作者：Steve McConnell | 类型：编程实践指南 | 评分：0.90

## 核心思维

**"软件构建是技术，也是艺术"**——McConnell认为编程不仅是写代码，更是解决问题的系统性方法。这本书涵盖了从变量命名到测试的所有软件构建实践。

## 底层逻辑思维

**结构化编程与软件质量**——书中内容基于大量的软件工程研究和实践经验，告诉你什么是真正有效的编程实践，而非主观建议。

## 核心行为准则

1. **质量优先**：代码首先是给人读的
2. **系统性方法**：编程是科学，不是艺术
3. **防御性编程**：为错误做准备
4. **持续重构**：代码需要不断优化

## 核心价值观

- **质量意识**：代码质量决定软件质量
- **经验传承**：从错误中学习，从成功中总结
- **实用主义**：理论要落地到实践
- **工匠精神**：代码值得精心打磨

## 应用场景

- 编程能力提升
- 软件工程实践
- 代码审查标准
- 团队编程规范制定

## 心力/情绪韧性

这本书不是讲高大上的架构设计，而是**脚踏实地的编程实践。** 它教你如何写出可维护、可读、高效的代码。这些是程序员的基本功，也是区分普通程序员和优秀程序员的关键。
