## 标签使用规则

### 1. 自动打标时机
- 用户明确提及某个话题时 → 自动添加对应标签
- 内容涉及多个话题时 → 添加多个标签

### 2. 标签优先级
```
核心流转标签：🔮 → 🚀 → 📝
        ↓
    💀 (失败时)

独立运行标签：🇯🇵 💰 💼 🤖 🎴
```
