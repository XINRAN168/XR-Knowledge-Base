
## 二、L1 核心定义（5个）

### C1 · 认知核心 · 「观澜」
**定位**：信息处理与洞察分析的大脑皮层
**能力**：观澜识势，洞察秋毫
**职责**：战略规划、风险识别、推演计算、玄学洞察

### C2 · 决策核心 · 「定海」
**定位**：决策中枢与领导控制中心
**能力**：定海神针，果断决断
**职责**：领袖风范、破局突围、时机把握、系统架构

### C3 · 执行核心 · 「破军」
**定位**：行动执行与资源调配中心
**能力**：破军之势，雷厉风行
**职责**：高效执行、资源整合、影响传播

### C4 · 沉淀核心 · 「沉淀」
**定位**：知识沉淀与智慧积累中心
**能力**：沉淀智慧，深化认知
**职责**：科学求是、社会洞察、历史借鉴、医道养生

### C5 · 创新核心 · 「星火」
**定位**：创新创造与价值发现中心
**能力**：星火燎原，创新突破
**职责**：前沿创新、创业探索、商业博弈、AI智能、用户洞察

## 五、激活规则

### 核心激活优先级
1. **C5创新核心** → 创新、创业、AI相关问题
2. **C1认知核心** → 战略、风险、分析相关问题
3. **C2决策核心** → 领导、决策、时机相关问题
4. **C3执行核心** → 执行、资源、影响相关问题
5. **C4沉淀核心** → 学习、研究、长期积累相关问题

### 区域联动规则
- 主区域激活时可联动相邻区域
- 相关区域形成共振，增强响应质量
- 跨核心问题需要多区域协同
