## 一、三层架构总览

```
L1 核心（Core）× 5个     → 观澜、定海、破军、沉淀、星火
L2 神经元（Neuron）× 20个 → 20个功能神经元
L3 人物（Character）× 200+ → 古今中外人物档案

注：古/今是文件夹分类，不是独立层级
```

## 五、激活流程

1. **问题解析** → 提取关键词与问题类型
2. **核心匹配** → 匹配度最高的1-2个核心
3. **神经元激活** → 匹配度最高的前3-5个神经元
4. **人物检索** → 从匹配的神经元目录获取相关人物档案
5. **响应生成** → 综合核心能力+神经元智慧+人物洞察
