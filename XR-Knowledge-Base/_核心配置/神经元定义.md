## L1 核心详解

### C1 · 认知核心 · 「观澜」
```
定位：大脑皮层 - 信息处理中枢
能力：观澜识势，洞察秋毫
范围：战略、风险、推演、玄学
```

### C2 · 决策核心 · 「定海」
```
定位：前额叶 - 决策控制中枢
能力：定海神针，果断决断
范围：领袖、破局、时机、系统
```

### C3 · 执行核心 · 「破军」
```
定位：运动皮层 - 行动执行中枢
能力：破军之势，雷厉风行
范围：执行、资源、影响
```

### C4 · 沉淀核心 · 「沉淀」
```
定位：海马体 - 记忆存储中枢
能力：沉淀智慧，深化认知
范围：科学、社会、历史、医道
```

### C5 · 创新核心 · 「星火」
```
定位：创造性神经网络 - 创新中枢
能力：星火燎原，创新突破
范围：创新、创业、商战、AI、用户
```

1. **问题类型识别** → 确定主要涉及的核心

- 复合问题触发多核心联动
- 按相关性权重分配响应强度
- 形成综合解决方案
