## 七、话题路由机制（V9.0）

### 7.1 路由流程

```
用户输入
    │
    ▼
【话题识别】MatchCalculator
    │  ├─→ 关键词提取
    │  ├─→ 问题类型分类
    │  └─→ 语义相似度计算
    │
    ▼
【话题路由】
    │  ├─→ 查询话题-神经元映射表
    │  ├─→ 计算激活优先级
    │  └─→ 生成激活列表
    │
    ▼
【四阶段碰撞】
    │  ├─→ 第0阶段：能量管理（按需激活）
    │  ├─→ 第一阶段：内部碰撞（多规则并行）
    │  ├─→ 第二阶段：之间碰撞（层级折叠）
    │  └─→ 第三阶段：记忆分割（涟漪存储）
    │
    ▼
【响应输出】
```

### 7.2 路由配置

```python
TOPIC_ROUTING_CONFIG = {
    "activation": {
        "max_primary_neurons": 3,
        "max_secondary_neurons": 2,
        "min_activation_score": 0.5,
        "activation_decay": 0.9
    },
    "collision": {
        "internal_parallel_rules": 3,
        "fold_threshold": 0.7,
        "segment_confidence": 0.8
    }
}
```
