## 三、八库联动机制

### 3.1 典型工作流

```
问题输入
    │
    ├──→ L05知识库 → 获取基础概念
    │
    ├──→ L04工具方法论库 → 获取分析框架
    │
    ├──→ L03场景案例库 → 获取案例参照
    │
    ├──→ L01_SOP流程库 → 标准化处理流程
    │
    ├──→ L06记忆库 → 查询上下文
    │
    └──→ L08项目库 → 查询项目状态

处理完成后
    │
    ├──→ L06记忆库 → 更新上下文
    │
    ├──→ L07学习库 → 沉淀学习成果
    │
    └──→ L08项目库 → 更新项目状态（如涉及）
```

### 3.2 反馈闭环

```
用户反馈
    │
    ├── 采纳 → L05知识库 + L07学习库
    │
    ├── 不采纳 → L04工具方法论库 + L07学习库
    │
    ├── 结果成功 → L05知识库 + L02专业领域库
    │
    └── 结果失败 → L03场景案例库（失败案例）+ L07学习库
```

## 四、使用规范

### 4.1 调用优先级

| 场景 | 优先调用 | 辅助调用 |
|:---|:---|:---|
| 新问题分析 | L05知识库 + L01_SOP流程库 | L04工具方法论库 |
| 案例参照 | L03场景案例库 | L02专业领域库 |
| 工具使用 | L04工具方法论库 | L07学习库 |
| 进度追踪 | L08项目库 | L06记忆库 |
| 知识学习 | L07学习库 | L05知识库 |

### 4.2 更新规范

- 新知识 → L05知识库
- 新方法 → L01_SOP流程库
- 新案例 → L03场景案例库
- 新工具 → L04工具方法论库
- 上下文变更 → L06记忆库
- 学习进展 → L07学习库
- 项目状态变更 → L08项目库
