## 一、话题分类体系（V9.0完整版）

> **重要区分**：
> - **T01-T10** = 问题类型（基于问题本质分类）
> - **S01-S11** = 业务场景话题（基于工作/生活场景分类）
> - 两者可叠加使用，例如：T01战略决策 + S03须弥推演

### 1.1 十大问题类型（T01-T10）

| 类型编号 | 类型名称 | 核心关键词 | 典型问题 | 激活特征 |
|:---:|:---|:---|:---|:---|
| **T01** | 战略决策 | 公司战略、发展方向、市场布局、竞争策略 | "公司明年应该往什么方向发展？" | 全力激活，第一阶段多人物并行 |
| **T02** | 竞争分析 | 竞争对手、优劣势、市场份额、差异化 | "如何看待XX公司的竞争优势？" | 中等激活，第二阶段重点交叉验证 |
| **T03** | 执行落地 | 执行计划、任务分解、资源分配、进度跟踪 | "如何确保项目按时完成？" | 轻量激活，第三阶段快速输出 |
| **T04** | 个人发展 | 职业规划、能力提升、人脉拓展、成长路径 | "我应该如何规划职业发展？" | 中等激活，涟漪存储扩展思考 |
| **T05** | 创业投资 | 创业项目、投资分析、商业模式、融资 | "这个创业项目值得投资吗？" | 全力激活，四阶段全流程 |

| **T07** | 团队领导 | 团队管理、激励机制、文化建设、人才梯队 | "如何打造高效团队？" | 中等激活，第二阶段重点验证 |
| **T08** | 产品市场 | 产品设计、用户需求、市场定位、增长策略 | "如何进行产品定位？" | 中等激活，第三阶段记忆分割 |
| **T09** | 危机处理 | 危机预警、应对策略、风险控制、舆情管理 | "遇到突发危机应该怎么办？" | 全力激活，四阶段全流程 |
| **T10** | 日常咨询 | 生活问题、学习方法、情感问题、健康养生 | "有什么高效学习方法推荐？" | 轻量激活，快速输出 |

## 二、话题识别机制

### 2.1 识别流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         话题识别流程                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  用户输入：                                                                  │
│  "公司面临市场激烈竞争，应该如何制定竞争策略？"                               │
│       │                                                                     │
│       ▼                                                                     │
│  【第一步】关键词提取                                                        │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │  分词 → 去停用词 → 关键词提取                                         │ │
│  │  结果：["公司", "市场", "竞争", "竞争策略", "制定"]                  │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│       │                                                                     │
│       ▼                                                                     │
│  【第二步】问题类型分类（T01-T10）                                          │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │  规则匹配 → T01战略决策(0.85) > T02竞争分析(0.72) > T05创业投资(0.3) │ │
│  │  结论：主要类型=T01，次要类型=T02                                     │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│       │                                                                     │
│       ▼                                                                     │
│  【第三步】业务场景识别（S01-S11）                                          │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │  关键词匹配 → S01情报·机会(0.6) → S03须弥推演(0.8)                   │ │
│  │  结论：触发场景=S03（需要推演验证）                                   │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│       │                                                                     │
│       ▼                                                                     │
│  【第四步】四阶段碰撞决策                                                    │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │  T01 + S03 → 全力激活 → 四阶段全流程                                 │ │
│  │  激活神经元：R01、R05、R08核心 + 辅助神经元                          │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 四、四阶段碰撞机制（V9.0核心）

### 4.1 碰撞流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         四阶段碰撞机制                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  话题识别 → 神经元激活                                                       │
│       │                                                                     │
│       ▼                                                                     │
│  【第0阶段】能量管理（按需激活）                                             │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │  • 评估问题复杂度                                                        │ │
│  │  • 分配计算资源                                                          │ │
│  │  • 决定参与神经元数量                                                    │ │
│  │  • 简单问题 → 轻量激活；复杂问题 → 全力激活                              │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│       │                                                                     │
│       ▼                                                                     │
│  【第一阶段】内部碰撞（多人物并行）                                          │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │  • 每个神经元内部多位人物独立思考                                         │ │
│  │  • 人物观点动态碰撞                                                       │ │
│  │  • 并行规则综合输出                                                       │ │
│  │  • 输出：各神经元独立结论                                                  │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│       │                                                                     │
│       ▼                                                                     │
│  【第二阶段】之间碰撞（层级折叠）                                            │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │  • 各神经元结论交叉验证                                                   │ │
│  │  • 相互质疑与补充                                                         │ │
│  │  • 层级折叠整合                                                           │ │
│  │  • 输出：综合判断                                                         │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│       │                                                                     │
│       ▼                                                                     │
│  【第三阶段】记忆分割（涟漪存储）                                            │
│  ┌───────────────────────────────────────────────────────────────────────┐ │
│  │  • 核心结论 → 短期记忆                                                    │ │
│  │  • 扩展思考 → 涟漪存储                                                    │ │
│  │  • 背景知识 → 知识库归档                                                   │ │
│  │  • 输出：结构化响应 + 记忆更新                                             │ │
│  └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
