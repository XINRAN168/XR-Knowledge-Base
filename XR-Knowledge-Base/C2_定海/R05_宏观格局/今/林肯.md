# 林肯 - 废除奴隶制的总统

## 基本信息
- **评分**: 0.93
- **时代**: 美国内战时期（1809-1865）
- **定位**: 美国第十六任总统，废除奴隶制、维护国家统一

## 五维度分析

### 1. 核心思维
**民有、民治、民享思维**
- 政府是人民的国家
- 一切人生而平等
- 维护联邦统一是最高职责
- 以斗争求统一

### 2. 底层逻辑思维
**道德制高点凝聚人心**
- 将内战定义为正义事业
- 解放奴隶作为战争目标
- 以 Gettysburg 演说升华战争意义
- 给予战败方和解机会

### 3. 核心行为准则
**坚定目标、灵活手段**
- 坚定废除奴隶制目标
- 战争期间暂停人身保护令
- 采取渐进式解放策略
- 重视民心和舆论

### 4. 核心价值观
**自由、平等、正义**
- 人人生而平等
- 联邦不可分裂
- 民主制度最优越
- 道义高于利益

### 5. 心力/情绪韧性
**在至暗时刻的坚持**
- 多次军事失败后的坚持
- 面对批评的隐忍
- 儿子病逝的悲痛
- 遇刺前的预感
