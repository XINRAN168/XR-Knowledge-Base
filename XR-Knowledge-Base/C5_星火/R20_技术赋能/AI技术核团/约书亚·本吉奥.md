# 约书亚·本吉奥 Yoshua Bengio

## 核心思维

**意识先验论**：注意力机制不仅是工程技巧，更可能是通向有意识AI的关键。提出了"意识先验"（Consciousness Prior）假说，认为高级认知涉及少数意识状态的低维表示，这是AI理解因果和泛化的关键。

**独立先验设计**：深度学习的成功依赖于数据，但真正的通用智能需要"独立于任务的先验知识"。如何让机器像人类一样从少数样本中学习，是连接主义的核心挑战。

## 底层逻辑思维

1. **因果推断**：从相关性到因果性的跨越是理解世界的关键
2. **表示学习**：好的表示应该是可解释的、可迁移的、可组合的
3. **注意力机制**：选择性注意是智能的核心操作
4. **元学习**：学会学习的能力比具体技能更重要

## 核心行为准则

- **学术纯粹性**：在商业化浪潮中保持对基础研究的投入
- **开放合作**：GAN发布后鼓励学术界广泛探索，而非申请专利
- **跨代培养**：对年轻研究者给予慷慨的指导和合作机会
- **社会责任感**：积极参与AI伦理讨论，推动负责任的AI发展

## 核心价值观

**好奇心驱动**：对智能本质的纯粹好奇心是研究的原动力。
**开放科学**：科学进步应该服务于全人类，而非成为商业壁垒。
**长期视野**：关注十年后的技术，不为短期利益牺牲长期目标。

## 心力/情绪韧性

| 维度 | 表现 |
|:---|:---|
| 抗压能力 | 在深度学习寒冬期坚持循环神经网络和注意力机制研究 |
| 韧性来源 | 对研究方向的深刻理解和对智能本质的追问 |
| 情绪管理 | 对学术竞争保持平和心态，更关注合作而非排名 |
| 恢复力 | 用突破性论文回应外界质疑，而非陷入辩论 |
