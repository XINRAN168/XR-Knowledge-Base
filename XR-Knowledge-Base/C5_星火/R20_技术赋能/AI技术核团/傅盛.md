# 傅盛

## 核心思维

**认知升级观**：人和人之间的本质差异是认知差异，而非技能差异。在AI时代，认知的边界决定了企业能走多远。猎豹移动从工具到内容的转型，核心是认知的突破。

**AI赋能战略**：AI不是行业，而是通用能力。通过将AI能力产品化，让更多应用具备智能化，是傅盛在AI时代的核心判断。

## 底层逻辑思维

1. **趋势判断**：技术浪潮中，方向比努力更重要
2. **聚焦突破**：在资源有限的情况下，选择一个点做深做透
3. **全球化视野**：从海外市场切入，避免国内红海竞争
4. **组织进化**：企业的瓶颈往往是创始人的瓶颈

## 核心行为准则

- **持续学习**：从程序员到CEO到AI布道者，每次转型都源于学习
- **公开分享**：通过演讲和文章分享思考，输出倒逼输入
- **冒险精神**：敢于进入未知领域，接受失败的可能
- **产品思维**：无论业务如何多元化，核心都是做出好产品

## 核心价值观

**认知驱动**：认知的边界决定了成长的边界。
**产品主义**：技术必须转化为用户价值，否则毫无意义。
**全球化公民**：不将自己局限在单一市场。

## 心力/情绪韧性

| 维度 | 表现 |
|:---|:---|
| 抗压能力 | 猎豹移动上市后的持续转型压力 |
| 韧性来源 | 不断学习带来的信心 |
| 情绪管理 | 面对外界质疑保持战略定力 |
| 恢复力 | 从工具出海到AI赛道的成功转型 |
