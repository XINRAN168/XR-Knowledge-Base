# 张亚勤

## 核心思维

**技术融合观**：5G、AI、云端、边缘计算的融合将重新定义计算范式。百度智能云的"云智一体"战略正是这一判断的落地——智能化必须与云基础设施深度整合。

**技术企业家精神**：科学家可以成为优秀的企业家，关键是将技术洞察转化为商业价值。技术是手段而非目的，服务用户才是目标。

## 底层逻辑思维

1. **系统性创新**：单一技术的突破往往难以形成持续优势，系统整合才能创造价值
2. **生态构建**：平台的价值来自生态伙伴的成功
3. **国际化布局**：技术创新必须具备全球视野
4. **产学研协同**：学术研究与产业应用相互促进

## 核心行为准则

- **技术愿景**：始终保持对前沿技术的敏锐度
- **人才培养**：建立有吸引力的人才体系
- **战略执行**：将愿景分解为可执行的路径
- **风险平衡**：创新与风险控制需要平衡

## 核心价值观

**技术改变世界**：相信技术有能力解决人类面临的重大挑战。
**长期投入**：核心技术需要十年维度的持续投入。
**开放合作**：技术创新需要开放生态的支撑。

## 心力/情绪韧性

| 维度 | 表现 |
|:---|:---|
| 抗压能力 | 在微软、百度等大企业的复杂环境中保持技术初心 |
| 韧性来源 | 对技术趋势的深度理解 |
| 情绪管理 | 面对行业变化保持冷静判断 |
| 恢复力 | 从学术到工业的成功转型 |
