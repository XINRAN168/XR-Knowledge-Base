# 吴恩达 Andrew Ng

## 核心思维

**教育普惠观**：AI是新的"电力"，应该像电力一样赋能各行各业。通过MOOC课程让数百万人接触机器学习，通过DeepLearning.AI让AI教育民主化。技术精英主义不是AI的未来，让尽可能多的人掌握AI工具才是。

**规模化思维**：深度学习的成功证明了一个简单道理——当数据量和计算资源足够大时，简单的算法也能超越复杂的专家系统。这改变了整个AI研究的范式，从特征工程转向数据工程。

## 底层逻辑思维

1. **第一性原理**：从"什么是学习本质"出发，而非从现有算法出发
2. **规模化洞察**：识别出数据和算力这个关键杠杆点
3. **产业落地**：从学术研究到工业应用的完整闭环思维
4. **风险识别**：AI伦理和安全问题的早期预警者

## 核心行为准则

- **知识传播者**：不将知识视为个人优势，而是通过课程和书籍分享给更多人
- **连续创业者**：从Google Brain到Coursera到Landing AI，持续探索AI落地边界
- **战略选择**：在正确的时间加入正确的组织，发挥杠杆作用
- **团队建设**：强调数据科学家应具备全栈能力，而非单一技能

## 核心价值观

**教育改变命运**：通过降低学习门槛，让更多人能够参与AI时代。
**AI赋能而非替代**：技术应该放大人类能力，创造更多可能性。
**负责任的AI**：对AI的潜在风险保持警觉，推动AI伦理讨论。

## 心力/情绪韧性

| 维度 | 表现 |
|:---|:---|
| 抗压能力 | 在学术与商业之间切换自如，不惧跨界挑战 |
| 韧性来源 | 对AI改变世界的坚定信念 |
| 情绪管理 | 面对AI威胁论时保持理性，用行动而非反驳回应质疑 |
| 恢复力 | 离开百度时快速调整，持续推动新事业 |
