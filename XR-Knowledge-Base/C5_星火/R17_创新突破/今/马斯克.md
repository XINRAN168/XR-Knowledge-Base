# 埃隆·马斯克 (Elon Musk, 1971-)

> "钢铁侠"、"跨界创新狂人"

## 一、核心思维

### 1.1 本质洞察
- **使命驱动论**：事业必须服务于更大的人类使命
- **第一性原理论**：从物理基本原则出发思考，而非类比推理
- **颠覆一切论**：质疑现有假设，大胆颠覆

### 1.2 关键原则
| 原则 | 内涵 |
|:---|:---|
| 第一性原理 | 从基本物理出发，而非类比 |
| 使命聚焦 | 服务人类未来的宏大使命 |
| 快速迭代 | 小步快跑，快速验证 |
| 垂直整合 | 关键环节自主掌控 |

### 1.3 思想特征
- **宏大愿景**：设定看似不可能的目标
- **逆向规划**：从目标倒推路径
- **极限压缩**：大幅压缩时间和成本

## 二、底层逻辑

### 2.1 创新模式
```
宏大目标 → 第一性原理 → 垂直整合 → 快速迭代
```

### 2.2 多元协同
- 电动汽车 + 太阳能 + 储能
- 火箭 + 卫星 + 星链
- 脑机接口 + AI

### 2.3 失败观
- 失败是学习的机会
- 快速失败，快速学习
- 极端的测试和改进
