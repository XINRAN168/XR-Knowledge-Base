# 彼得·蒂尔 (Peter Thiel, 1967-)

> "PayPal创始人"、"硅谷创投教父"

## 一、核心思维

### 1.1 本质洞察
- **从0到1论**：真正的创新是从无到有，而非复制
- **竞争论**：竞争是失败者的游戏，垄断才是目标
- **逆向思维论**：做没人认同的事才能创造价值

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 从0到1 | 垂直创新，创造新事物 |
| 从1到N | 水平复制，竞争对手竞争 |
| 垄断企业 | 拥有独特价值的公司 |
| 秘密 | 尚未被发现的真理 |

### 1.3 思想特征
- **逆向思维**：与多数人相反
- **长期视角**：关注未来趋势
- **精英主义**：相信少数人能改变世界

## 二、底层逻辑

### 2.1 创新逻辑
```
从0到1 → 技术进步 → 指数增长
从1到N → 复制模仿 → 竞争红海
```

### 2.2 垄断机制
- 专利技术
- 网络效应
- 规模经济
- 品牌

### 2.3 创业成功
- 发现秘密
- 找到切入点
- 规模化扩张
