# 大前研一 (Kenichi Ohmae, 1943-)

> "日本战略之父"、"麦肯锡传奇"

## 一、核心思维

### 1.1 本质洞察
- **战略三要素**：战略=客户利益+公司能力+竞争差异化
- **思考力论**：新经济时代，思考力差距化造成经济能力差距
- **洞悉本质**：能够洞悉问题本质的人与无法洞悉的人之间存在巨大差距

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 逻辑思考 | 以事实和数据为基础的理性思考 |
| 非线性思维 | 跳出一因一果的简单逻辑 |
| MECE原则 | 相互独立，完全穷尽 |
| 金字塔结构 | 结论先行，证据支撑 |

### 1.3 思想特征
- **跨国视野**：超越国界思考企业战略
- **顾客中心**：战略起点是顾客而非产品
- **实操导向**：注重思考方法的可执行性

## 二、底层逻辑

### 2.1 商业洞察框架
```
观察现象 → 假设原因 → 实地验证 → 得出结论
```

### 2.2 看不见的新大陆
- 实体经济：无国界经济的延伸
- 现金流与信息流：自由跨境流动
- 数字经济：互联网驱动的经济形态
- 倍数经济：资金以百倍千倍规模流动

### 2.3 思考力本质
- 商业感觉：理解和把握商业本质
- 分析能力：将复杂问题分解
- 决策勇气：基于判断采取行动

## 三、行为准则

### 3.1 思考技术实践
| 准则 | 应用 |
|:---|:---|
| 假设-验证 | 不做不知道，提想法必须验证 |
| 现场主义 | 实地访谈比数据分析更重要 |
| 金字塔结构 | 一页一个结论，论据支撑 |
| 非线性思维 | 跳出因果链思考 |

### 3.2 提案技巧
- 只给一个核心建议
- 用数据支撑结论
- 简洁表达45分钟内完述
- 洞悉对方心理状态

### 3.3 商业判断标准
- 能否洞悉本质？
- 是否符合逻辑？
- 能否被采纳执行？

## 四、价值观

### 4.1 商业伦理观
- 对事不对人
- 坚持事实基础
- 勇于否定自己

### 4.2 组织观
- 优秀企业有否定自己的勇气
- 吸收异质特质融入文化
- 变革力量来自内部

### 4.3 全球视野
- 超越国家框架思考
- 全球化资源配置
- 本地化运营执行

## 五，心力韧性

### 5.1 职业生涯
| 阶段 | 特点 |
|:---|:---|
| 麦肯锡23年 | 从项目经理到全球董事 |
| 独立顾问 | 为政府和企业提供战略咨询 |

- 促进了麦肯锡方法在亚洲的传播
