# 亨利·明茨伯格 (Henry Mintzberg, 1939-)

> "管理实践主义大师"、"战略管理学派批判者"

## 一、核心思维

### 1.1 本质洞察
- **管理即手艺**：管理者是工匠，战略是他们的黏土
- **战略涌现论**：只有10%-30%的战略能完全按计划实现，大部分战略是涌现的
- **结构决定论**：组织结构影响战略，反之亦然

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 战略三类型 | 意图战略、已实现战略、涌现战略 |
| 管理者十角色 | 人际、信息、决策三大类共十种角色 |
| 战略形成学派 | 设计、计划、定位、企业家等十大流派 |
| 五种组织结构 | 简单、机械、专业、分权、灵活 |

### 1.3 思想特征
- **批判精神**：挑战传统管理教育的象牙塔
- **实践导向**：强调从经验中学习
- **整合视野**：综合多学派视角理解战略

## 二、底层逻辑

### 2.1 管理者工作现实
- **时间碎片化**：工作被打断，缺乏整块时间
- **即时性**：偏好口头而非书面沟通
- **选择性**：关注当前、具体、熟悉的事务

### 2.2 战略形成机制
```
意图战略 ──→ 涌现战略 ──→ 已实现战略
   │                              ↑
   └─── 未实现战略 ←──────────────┘
```

### 2.3 组织结构类型
| 结构 | 特征 | 适用 |
|:---|:---|:---|
| 创业型 | 简单、直接 | 小型初创 |
| 机械型 | 标准化、层级化 | 大规模生产 |
| 专业型 | 去中心化、技能标准化 | 医院、大学 |
| 分部型 | 事业部自治 | 多元化企业 |
| 灵活型 | 项目导向、无层级 | 创新组织 |

## 三、行为准则

### 3.1 管理者行动指南
- **深入现场**：保持与业务实际的联系
- **善用直觉**：在分析基础上信任直觉
- **平衡稳定与变革**：长周期进化与适时革命结合
- **战略学习**：从成功与失败中学习

### 3.2 战略思考原则
- 战略不是规划出来的，而是在实践中涌现的
- 倾听一线员工的反馈比市场调研更重要
- 稳定性是演化的前提，变革是跳跃的准备

### 3.3 组织设计要点
- 关注协调机制而非仅组织图
- 鼓励非正式沟通
- 允许适度的混乱

## 四、价值观

### 4.1 管理哲学
- 管理是一门手艺，需要实践智慧
- 不存在"最佳管理方式"，一切取决于情境
- 管理教育应该培养实践者而非分析师

### 4.2 组织观
- 组织是有机的生命体，不是机器
- 尊重组织中的非正式力量
- 关注基层创新

### 4.3 社会责任
- 反对将商业逻辑泛化到所有领域
- 关注管理对社会的影响
- 提倡负责任的商业实践

## 五，心力韧性

### 5.1 学术独立精神
- 长期批判主流管理教育
- 拒绝学术界与实践的脱节
- 坚持自己的独立判断

### 5.2 经典语录
> "管理者是工匠，战略是他们的黏土。"

> "战略规划不是战略思考。战略规划往往破坏了战略思考。"

> "管理是一种实践，其本质在于综合，它不是科学，也不是职业，而是手艺。"

### 5.3 影响力
- 影响了管理教育从理论导向转向实践导向
- 促进了组织理论的发展
- 挑战了MBA教育的传统模式
