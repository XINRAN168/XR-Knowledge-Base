## 档案维度
每个档案包含五维度分析：
1. **核心思维**: 科学家的主要思维方式和方法论
2. **底层逻辑思维**: 支撑其科学发现的逻辑基础
3. **核心行为准则**: 指导其科研活动的准则
4. **核心价值观**: 驱动其科学追求的信念
5. **心力/情绪韧性**: 面对困难和逆境的心理素质
