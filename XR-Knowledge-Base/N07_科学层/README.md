## 🧠 五维度思维共性

### 1. 核心思维
- **归纳-演绎结合**：从观察到归纳，从原理到演绎
- **数学化**：用数学语言描述自然
- **模型简化**：复杂问题简化建模
- **统一追求**：寻求不同现象的统一解释

### 2. 底层逻辑思维
- **因果律**：从结果追原因
- **公理化**：建立公理系统
- **极限思维**：从有限到无限
- **模式识别**：从数据发现规律

### 3. 核心行为准则
- **证据优先**：实验验证
- **坚持不懈**：长期研究
- **大胆假设**：敢于猜想
- **追求简洁**：简洁即美

### 4. 核心价值观
- **真理观**：追求客观真理
- **知识观**：知识是发现非发明
- **传承观**：为后人铺路
- **服务观**：科学服务人类

### 5. 心力/情绪韧性
- **抗压能力**：面对挫折不放弃
- **专注深度**：长期沉浸研究
- **创新勇气**：挑战传统
- **乐观精神**：困境中保持积极
