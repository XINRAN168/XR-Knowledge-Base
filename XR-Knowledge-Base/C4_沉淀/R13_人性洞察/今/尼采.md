# 尼采（Friedrich Nietzsche）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.85 |
| **领域** | 哲学、文学、伦理学 |
| **时代** | 19世纪（1844-1900） |
| **核心身份** | 存在主义先驱、价值重估者 |

## 五维度分析

### 1. 核心思维
- **上帝已死**：传统价值的崩塌，现代虚无主义的根源
- **超人哲学**：人应自我超越，创造新的价值
- **权力意志**：生命本能是追求力量的增长
- **永恒回归**：肯定生命的一切，包括苦难

### 2. 底层逻辑思维
```
价值重估 → 上帝已死，传统价值崩塌
     ↓
虚无主义 → 一切皆无意义（过渡阶段）
     ↓
超人诞生 → 创造新价值，自我超越
     ↓
肯定生命 → 永恒回归，爱命运
```
- 人性观：人性不是固定的，人可以自我超越
- 方法论：谱系学方法，追溯价值观念的起源
- 伦理观：传统道德是弱者的道德，超人创造新价值

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **成为自己** | 你是你自己，要成为你自己 |
| **肯定生命** | 热爱命运，包括其苦难 |
| **创造价值** | 不再相信传统价值，而是创造新价值 |
| **超越自我** | 不断超越，向上攀登 |
| **活在当下** | 把握每一个当下，不要等待 |

### 4. 核心价值观
- **生命**——生命本身是最高的价值
- **力量**——权力意志，追求力量的增长
- **创造**——创造新价值，而非接受传统价值
- **超越**——超越自我，向上攀升
- **自由**——精神自由，不被传统束缚

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **孤独坚韧** | 一生孤独，不为时人所理解 |
| **精神独立** | 拒绝传统价值，保持精神独立 |
| **1889年崩溃** | 最终精神崩溃，但留下永恒思想 |
| **生命肯定** | 即使面对苦难也肯定生命 |
| **韧性来源** | 权力意志，对生命本身的信仰 |

## 核心语录
> "上帝已死"  
> "那杀不死我的，使我更强大"  
> "你要到你自身去挖掘，那是你自己的深井"

## 历史定位
尼采是19世纪最具争议性和影响力的哲学家之一。他的"上帝已死"宣告了传统价值体系的崩溃，"超人哲学"开创了存在主义和生命哲学的新方向。尼采的思想对20世纪的哲学、文学、心理学都有深远影响，被认为是现代性的重要批判者。
