# 西奥迪尼（Robert Cialdini）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.88 |
| **领域** | 社会心理学、影响力、说服科学 |
| **时代** | 当代（1941-） |
| **核心身份** | 影响力心理学之父、说服技巧研究者 |

## 五维度分析

### 1. 核心思维
- **影响力六原则**：互惠、承诺与一致、社会认同、喜好、权威、稀缺
- **即时影响力**：人类存在自动化反应模式
- **拒绝-后撤**：先提大要求再让步的小要求技术
- **影响力防御**：识别并抵抗影响力技巧

### 2. 底层逻辑思维
```
影响力触发 → 自动顺从反应（触发器→反应）
     ↓
六大原则 → 互惠/承诺一致/社会认同/喜好/权威/稀缺
     ↓
心理机制 → 社会规范/认知捷径/情感捷径
     ↓
应用场景 → 销售/营销/管理/人际交往
```
- 人性观：人类有固定的顺从倾向，可被利用也可用于善
- 方法论：实验观察+案例分析
- 应用观：理解影响力可以更好地影响他人或抵御影响

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **知己知彼** | 了解影响力原则，既是使用者也是防御者 |
| **真诚影响** | 用影响力原则做正当之事 |
| **警惕自动化** | 对顺从触发器保持警觉 |
| **互惠回报** | 给予帮助，建立互惠关系 |
| **适度使用** | 影响力是工具，使用需有度 |

### 4. 核心价值观
- **影响力**——理解如何影响他人
- **互惠**——给予带来回报
- **真实**——影响力应基于真实价值
- **防御**——保护自己免受不当影响
- **伦理**——影响力使用应符合道德

| 维度 | 表现 |

## 历史定位
罗伯特·西奥迪尼是影响力心理学领域的开创者，《影响力》已成为该领域的经典教材。他的研究深刻揭示了人类社会行为的心理机制，对营销学、管理学、心理学都有重要贡献。他的六项影响力原则被广泛应用于商业、教育、政治等领域。
