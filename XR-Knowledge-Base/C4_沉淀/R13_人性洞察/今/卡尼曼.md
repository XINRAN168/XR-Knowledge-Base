# 卡尼曼（Daniel Kahneman）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.90 |
| **领域** | 行为经济学、认知心理学、决策科学 |
| **时代** | 当代（1934-） |
| **核心身份** | 行为经济学之父、诺贝尔经济学奖得主 |

## 五维度分析

### 1. 核心思维
- **双系统理论**：系统1（快速直觉）/系统2（缓慢理性）
- **前景理论**：人们对损失的反应比对等量收益强2.5倍
- **认知偏见**：锚定、可得性、代表性等系统性判断偏差
- **噪声**：决策中的随机变异，与偏见同等重要

### 2. 底层逻辑思维
```
认知系统 → 系统1（快、自动、情绪化）/ 系统2（慢、费力、理性）
     ↓
决策偏差 → 偏见（系统性错误）+ 噪声（随机变异）
     ↓
心理账户 → 人们如何主观建构决策问题
     ↓
行为规律 → 前景理论：价值函数、概率权重
```
- 人性观：人类决策充满系统性偏差，非理性是常态
- 方法论：实验经济学+认知心理学结合
- 应用观：理解偏见可以帮助做出更好的决策

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **慢想慎行** | 重要决策应调动系统2慢慢思考 |
| **警惕直觉** | 对自己的直觉保持怀疑 |
| **防噪设计** | 在决策中引入结构化方法减少噪声 |
| **外部视角** | 考虑外部视角（基准率）而非仅内部信息 |
| **预见偏差** | 决策时预见到可能的偏见 |

### 4. 核心价值观
- **理性**——追求理性决策
- **谦逊**——承认认知局限
- **实证**——用实验检验假设
- **精确**——追求决策的精确性
- **改进**——通过理解偏差改进决策

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术跨界** | 心理学家获得诺贝尔经济学奖 |
| **晚年成就** | 60岁后获得诺贝尔奖，仍持续研究 |
| **理性自省** | 用理性研究非理性的认知偏差 |
| **谦逊态度** | 对人类认知能力持谦逊态度 |
| **韧性来源** | 对人类理性的复杂理解**

## 核心语录
> "我们看到的不是世界本身，而是我们对世界的看法"  
> "在一个人的生活中，最重要的决定是谁将成为你的主治医生"  
> "噪声是决策中的随机变异，偏见是系统性错误"

## 历史定位
丹尼尔·卡尼曼是将心理学与经济学融合的先驱，他开创的行为经济学彻底改变了我们对人类决策的理解。获得诺贝尔经济学奖标志着他的贡献得到主流学术界的认可。《思考，快与慢》成为决策科学领域的里程碑著作。
