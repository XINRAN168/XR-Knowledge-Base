# 马斯洛（Abraham Maslow）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.85 |
| **领域** | 人本主义心理学、动机理论 |
| **时代** | 20世纪（1908-1970） |
| **核心身份** | 人本主义心理学之父、需求层次理论提出者 |

## 五维度分析

### 1. 核心思维
- **需求层次**：生理→安全→归属→尊重→自我实现
- **自我实现**：人类最高需求，充分发挥潜能
- **高峰体验**：自我实现时刻的巅峰体验
- **人本主义**：研究健康人而非病人

### 2. 底层逻辑思维
```
需求层次 → 生理→安全→归属→尊重→自我实现
     ↓
层次递进 → 低层满足后产生高层需求
     ↓
自我实现 → 充分发挥潜能，实现理想自我
     ↓
高峰体验 → 自我实现时刻的忘我状态
```
- 人性观：人性本善，有自我实现的倾向
- 方法论：研究优秀个体（自我实现者）而非病人
- 发展观：人是可以不断成长和发展的

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **自我实现** | 充分发挥自己的才能和潜能 |
| **高峰体验** | 追求忘我投入的巅峰体验 |
| **积极人性** | 关注人的积极品质和潜能 |
| **成长导向** | 人生是不断成长和发展的过程 |
| **潜能开发** | 每个人都有未开发的潜能 |

### 4. 核心价值观
- **自我实现**——人生的最高目标
- **潜能**——每个人都有未实现的潜能
- **积极**——关注人性积极面
- **成长**——人是可以不断成长的
- **高峰**——追求高峰体验

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术独立** | 开创与行为主义、精神分析并立的人本主义 |
| **勇于创新** | 挑战主流心理学研究取向 |
| **全面发展** | 涉猎广泛，不仅心理学还研究管理、教育等 |
| **早逝未竟** | 1970年去世，留下未完成的超人心理学 |
| **韧性来源** | 对人类潜能的积极信念**

## 核心语录
> "一个人能够成为什么，他就必须成为什么"  
> "自我实现者专注于问题而非自我"  
> "如果你只有锤子，你会把所有问题都看成钉子"

## 历史定位
马斯洛是人本主义心理学的创始人之一，他的需求层次理论成为20世纪最有影响力的心理学理论之一。他开创性地研究健康的自我实现者而非心理病人，对积极心理学、管理学、领导力理论都有重要贡献。
