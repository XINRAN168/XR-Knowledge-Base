# 荀子（Xunzi）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.88 |
| **领域** | 儒家思想、伦理学、政治哲学 |
| **时代** | 战国末期（约前313-前238） |
| **核心身份** | 儒家集大成者、礼法思想奠基人 |

## 五维度分析

### 1. 核心思维
- **性恶论**：人性本恶，其善者伪也（人为）
- **礼法并施**：隆礼重法，礼乐刑政相辅相成
- **劝学思想**：学不可以已，青出于蓝而胜于蓝
- **制天役物**：大天而思之，孰与物畜而制之

### 2. 底层逻辑思维
```
人本性恶 → 好利疾恶，耳好声，目好色
     ↓
道德起源 → 圣王教化，礼义法度（伪）
     ↓
社会秩序 → 礼以定伦，分等级，明秩序
     ↓
治国之道 → 隆礼尊贤，法治与教化并行
```
- 人性观：性恶论，人有天然的好利欲，需要外部约束
- 方法论：通过学习、师法、礼义来改变本性
- 政治观：礼法并用，以礼为主，以法为辅

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **隆礼重法** | 故学至乎礼而止矣，夫是之谓道德之极 |
| **学不可以已** | 吾尝终日而思矣，不如须臾之所学也 |
| **锲而不舍** | 锲而舍之，朽木不折；锲而不舍，金石可镂 |
| **君子自强** | 君子敬其在己者，而不慕其在天者 |
| **积善成德** | 积土成山，风雨兴焉；积善成德，而神明自得 |

### 4. 核心价值观
- **礼**——社会秩序的根本，人伦之纲纪
- **义**——行为的正当性标准
- **法**——强制性的社会规范
- **学**——改变命运的途径
- **群**——人能群，彼不能群

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **理性冷静** | 性本恶论，对人性持冷静客观的态度 |
| **务实进取** | 大天而思之，制天役物而非听天由命 |
| **自强不息** | 君子敬其在己者，而不慕其在天者 |
| **积累精神** | 积土成山，积善成德，强调积累的力量 |
| **韧性来源** | 理性认知人性，通过后天努力改变命运 |

## 核心语录
> "人性本恶，其善者伪也"  
> "学不可以已"  
> "青，取之于蓝而青于蓝；冰，水为之而寒于水"

## 历史定位
荀子是先秦儒家的集大成者，他继承并发展了儒家思想，同时批判吸收了道家、法家思想。他打破孔孟"法先王"的传统，主张"法后王"，具有现实主义色彩。荀子的礼法思想对后世儒法合流有重要影响，韩非、李斯皆出其门下。
