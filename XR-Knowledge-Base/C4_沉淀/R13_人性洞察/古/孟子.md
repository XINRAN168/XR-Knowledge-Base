# 孟子（Mencius）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.90 |
| **领域** | 儒家思想、伦理学、政治哲学 |
| **时代** | 战国中期（约前372-前289） |
| **核心身份** | 亚圣、儒家思孟学派代表 |

## 五维度分析

### 1. 核心思维
- **性善论**：人性本善，恻隐之心人皆有之
- **仁政学说**：民为贵，社稷次之，君为轻
- **义利之辨**：何必曰利？亦有仁义而已矣
- **养气论**：吾善养吾浩然之气

### 2. 底层逻辑思维
```
人本性善 → 四端之心（恻隐、羞恶、辞让、是非）
     ↓
道德自觉 → 扩充善端（求则得之，舍则失之）
     ↓
人格修养 → 养浩然之气（至大至刚，塞于天地之间）
     ↓
社会理想 → 仁政王道（以德行仁者王）
```
- 人性观：性善论，认为人有先天的道德萌芽
- 方法论：反求诸己，从内心出发扩充善端
- 政治观：民本思想，统治者应以民意为依归

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **反求诸己** | 行有不得者皆反求诸己，其身正而天下归之 |
| **舍生取义** | 生，亦我所欲也；义，亦我所欲也；二者不可得兼，舍生而取义者也 |
| **不动心** | 我四十不动心；告子先我不动心 |
| **威武不屈** | 富贵不能淫，贫贱不能移，威武不能屈 |
| **乐以天下** | 乐以天下，忧以天下 |

### 4. 核心价值观
- **仁**——恻隐之心，仁之端也
- **义**——羞恶之心，义之端也
- **礼**——辞让之心，礼之端也
- **智**——是非之心，智之端也
- **道**——舍生取义，杀身成仁

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **使命担当** | 如欲平治天下，当今之世，舍我其谁也 |
| **逆境坚守** | 天将降大任于斯人也，必先苦其心志 |
| **浩然正气** | 其为气也，至大至刚，以直养而无害 |
| **自我确信** | 万物皆备于我，反身而诚，乐莫大焉 |
| **韧性来源** | 对人性善的信念，对使命的担当 |

## 核心语录
> "民为贵，社稷次之，君为轻"  
> "天将降大任于斯人也，必先苦其心志，劳其筋骨"  
> "富贵不能淫，贫贱不能移，威武不能屈"

## 历史定位
孟子是儒家学派的重要继承者和发展者，被后世尊为"亚圣"。他系统化了孔子的仁学思想，提出性善论和仁政学说，构建了完整的儒家政治哲学。孟子的"民本"思想对后世的政治批判传统有深远影响，其"大丈夫"人格理想成为士人的精神标杆。
