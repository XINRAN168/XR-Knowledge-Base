# 墨子（Mozi）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.85 |
| **领域** | 墨家思想、逻辑学、科技思想 |
| **时代** | 战国初期（约前470-前391） |
| **核心身份** | 墨家学派创始人、平民哲学倡导者 |

## 五维度分析

### 1. 核心思维
- **兼爱**：兼相爱，交相利，无差别之爱
- **非攻**：反对不义战争，爱好和平
- **尚贤**：官无常贵，民无终贱，选贤与能
- **节用**：节葬节用，批判儒家礼制浪费

### 2. 底层逻辑思维
```
人类困境 → 天下之乱，起于不相爱
     ↓
解决之道 → 兼以易别，以兼易别
     ↓
实现路径 → 兴天下之利，除天下之害
     ↓
检验标准 → 三表法（上本之古者圣王之事，下原察百姓耳目之实，发以为刑政）
```
- 人性观：功利主义，认为人天生有趋利避害的本能
- 方法论：三表法，重视经验和实效
- 社会观：平等博爱，反对儒家等级制度

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **兼爱** | 视人之国若视其国，视人之家若视其家，视人之身若视其身 |
| **非攻** | 亏人自利，义不义；入人园圃，窃人桃李，众闻则非之 |
| **尚同** | 上之所是必皆是之，上之所非必皆非之 |
| **节用** | 凡足以奉给民用则止，诸加费不加于民利者，圣王弗为 |
| **非命** | 强则治，弱则乱；非命论，反对宿命论 |

### 4. 核心价值观
- **兼**——无差别的平等之爱
- **利**——天下之利，公利为先
- **义**——兴天下之利，除天下之害
- **贤**——选贤举能，不问出身
- **勤**——强本节用，自强不息

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **道德勇气** | 摩顶放踵以利天下，虽杀身成仁而不辞 |
| **意志坚定** | 止楚攻宋，体现墨家的坚定信念 |
| **务实行动** | 不仅坐而言，更起而行，身体力行 |
| **自我牺牲** | 赴汤蹈火，死不旋踵 |
| **韧性来源** | 对"义"的信仰和对天下苍生的关怀 |

## 核心语录
> "兼相爱，交相利"  
> "兴天下之利，除天下之害"  
> "志不强者智不达，言不信者行不果"

## 历史定位
墨子是战国时期与儒家并称的"显学"代表，他代表平民阶层的声音，提出与儒家截然不同的社会理想。墨子的"兼爱"思想具有原始的平等意识，"非攻"思想体现了和平主义倾向。墨家还重视逻辑学和科技发明，墨家在先秦时期影响巨大，但汉代以后逐渐衰落。
