# 庄子（Zhuangzi）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.90 |
| **领域** | 道家思想、哲学、文学 |
| **时代** | 战国中期（约前369-前286） |
| **核心身份** | 道家学派代表、逍遥游精神开创者 |

## 五维度分析

### 1. 核心思维
- **逍遥境界**：乘天地之正，御六气之辩，以游无穷
- **齐物论**：万物齐一，是非无定，天地一指也
- **无用之用**：人皆知有用之用，而莫知无用之用
- **天籁自然**：天籁自鸣，不假外求

### 2. 底层逻辑思维
```
宇宙本体 → 道通为一，万物皆一
     ↓
认识论 → 是非相对，莫若以明
     ↓
人生观 → 逍遥自在，无待而游
     ↓
处世智慧 → 外化而内不化，随物应化
```
- 人性观：人性自然，返璞归真
- 方法论：无为自然，不刻意追求
- 生死观：方生方死，方死方生，生死如一

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **无用之用** | 人皆知有用之用，而莫知无用之用也 |
| **外化内不化** | 外化而内不化，与世沉浮而不同流合污 |
| **丧我坐忘** | 堕肢体，黜聪明，离形去知，同于大通 |
| **顺其自然** | 依乎天理，因其固然，顺物自然 |
| **物物不物于物** | 物物而不物于物，则胡可得而累邪 |

### 4. 核心价值观
- **真**——返璞归真，保全真我
- **游**——逍遥游，精神自由
- **自然**——顺其自然，不加人为
- **超越**——超越是非、得失、生死
- **自性**——各适其性，各安其命

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **精神超脱** | 举世而誉之而不加劝，举世而非之而不加沮 |
| **对死亡的超然** | 夫大块载我以形，劳我以生，佚我以老，息我以死 |
| **内心自由** | 若夫乘天地之正，而御六气之辩，以游无穷 |
| **超越功利** | 至人无己，神人无功，圣人无名 |
| **韧性来源** | 与道合一的精神境界，万物齐一的宇宙观 |

## 核心语录
> "北冥有鱼，其名为鲲。鲲之大，不知其几千里也"  
> "人皆知有用之用，而莫知无用之用也"  
> "天地与我并生，而万物与我为一"

## 历史定位
庄子是道家思想的杰出代表，他发展了老子思想中的自由精神，开创了"逍遥游"的精神境界。庄子以其独特的文学风格和哲学智慧，对中国文学、艺术、宗教都有深远影响。庄子的"齐物论"和"逍遥游"成为中国人精神超脱的重要思想资源。
