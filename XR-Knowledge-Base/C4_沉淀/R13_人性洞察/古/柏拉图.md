# 柏拉图（Plato）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.88 |
| **领域** | 哲学、政治学、形而上学 |
| **时代** | 古希腊（前428-前348） |
| **核心身份** | 理念论创始人、雅典学院创立者 |

## 五维度分析

### 1. 核心思维
- **理念论**：现实世界是理念世界的影子，理念是永恒不变的实在
- **理想国**：哲学家王，正义之城，各司其职
- **回忆说**：知识是灵魂对理念世界的回忆
- **洞穴隐喻**：人类如同囚于洞穴，只能看到墙上的影子

### 2. 底层逻辑思维
```
宇宙本体 → 理念世界（真实存在）
     ↓
认识论 → 回忆说（知识即回忆）
     ↓
人性论 → 灵魂三分（理性、激情、欲望）
     ↓
政治论 → 理想国（哲学家王）
```
- 人性观：灵魂不朽，肉体是灵魂的监狱
- 方法论：辩证法，从具体到抽象的不断提升
- 政治观：理想政治由哲学家治理

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **追求真理** | 哲学是对死亡的准备，追求理念世界的真善美 |
| **正义至上** | 正义不仅是外在行为，更是灵魂的和谐 |
| **教育为本** | 教育的最高目标是使灵魂转向 |
| **德性智慧** | 智慧是最高的美德，统摄其他美德 |
| **节制生活** | 战胜欲望，保持灵魂的平衡 |

### 4. 核心价值观
- **理念**——永恒不变的实在，美善之源
- **真理**——对理念世界的认识
- **正义**——各司其职，秩序和谐
- **智慧**——最高的善，理性的运用
- **美**——对理念世界的追求

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **精神独立** | 不受时局影响，坚持自己的哲学理想 |
| **教育坚守** | 创办雅典学院，传播哲学思想四十年 |
| **政治抱负** | 三叙杰拉什，实践理想国理念 |
| **思想勇气** | 挑战时俗，提出惊世骇俗的政治主张 |
| **韧性来源** | 对理念世界的信仰，对永恒真理的追求 |

## 核心语录
> "理念是永恒不变的实在"  
> "哲学家王，哲人为统治者"  
> "正义不仅是外在行为，更是灵魂的和谐"

## 历史定位
柏拉图是西方哲学史上最有影响力的思想家之一，他创立的理念论成为西方哲学的核心传统。他创办的雅典学院持续了近千年之久，培养了亚里士多德等大批哲学家。柏拉图的政治哲学，特别是《理想国》对后世政治思想有深远影响。
