# 亚里士多德（Aristotle）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.88 |
| **领域** | 哲学、逻辑学、科学、政治学、伦理学 |
| **时代** | 古希腊（前384-前322） |
| **核心身份** | 百科全书式学者、西方科学奠基人 |

## 五维度分析

### 1. 核心思维
- **形而上学**：实体论，一切存在都是个别实体
- **中庸之道**：德性是两个极端之间的中道
- **形式逻辑**：三段论，演绎推理的奠基人
- **目的论**：自然界一切都有目的（telos）

### 2. 底层逻辑思维
```
存在论 → 四因说（质料、形式、动力、目的）
     ↓
认识论 → 归纳与演绎结合
     ↓
伦理学 → 中庸之道，过犹不及
     ↓
政治学 → 人是政治的动物，城邦是自然存在
```
- 人性观：人是理性的动物，追求幸福是最终目的
- 方法论：经验观察与逻辑推理并重
- 伦理观：中道为德，德性是习惯的产物

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **中庸之道** | 德性是两种恶之间的中道，恶是过度或不足 |
| **德性养成** | 德性既非出于本性，也非反乎本性，而是出于习惯 |
| **理性生活** | 理性是人的本质，活出理性是人生最高目的 |
| **幸福追求** | 幸福是灵魂合乎德性的活动 |
| **实践智慧** | 理论智慧与实践智慧的结合 |

### 4. 核心价值观
- **中庸**——德性在两个极端之间
- **理性**——人的本质是理性
- **幸福**——人生的最高目标是幸福（eudaimonia）
- **德性**——美好生活的品质
- **实践**——在行动中实现德性

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术严谨** | 博学多才，建立系统的学科体系 |
| **理性平衡** | 追求中道，在极端之间保持平衡 |
| **流亡自持** | 被迫离开雅典，仍保持学术追求 |
| **百科全书** | 涉猎广泛，著述宏富 |
| **韧性来源** | 对理性生活的信仰，对知识的不懈追求 |

## 核心语录
> "人是政治的动物"  
> "德性是两种恶之间的中道"  
> "幸福是灵魂合乎德性的活动"

## 历史定位
亚里士多德是古希腊最博学的思想家，他几乎在每个知识领域都做出了开创性贡献。他创立的形式逻辑支配了西方思想两千多年，他的著作成为中世纪经院哲学的基础。亚里士多德的影响如此深远，以至于他的思想一度被视为权威，直到文艺复兴和科学革命才被重新审视。
