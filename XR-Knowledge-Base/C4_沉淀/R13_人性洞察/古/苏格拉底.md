# 苏格拉底（Socrates）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.90 |
| **领域** | 哲学、伦理学、教育学 |
| **时代** | 古希腊（前469-前399） |
| **核心身份** | 西方哲学奠基人、对话法创始人 |

## 五维度分析

### 1. 核心思维
- **认识你自己**：自知其无知是智慧的开端
- **德性即知识**：无人自愿作恶，作恶源于无知
- **产婆术**：通过对话、提问激发对方自我思考
- **普遍定义**：追求概念的本质定义而非个别实例

### 2. 底层逻辑思维
```
人性本质 → 理性灵魂，追求真理
     ↓
道德基础 → 德性可教，德性即知识
     ↓
认识论 → 承认无知，才能获得真知
     ↓
教育法 → 产婆术，唤醒内在知识
```
- 人性观：人是有理性的存在，灵魂是不朽的
- 方法论：对话法，通过提问引导而非直接灌输
- 伦理观：美德是灵魂的健康与和谐

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **自知其无知** | 我唯一知道的，就是我一无所知 |
| **德性可教** | 美德即是知识，愚昧是罪恶之源 |
| **认识你自己** | 未经审视的人生不值得过 |
| **对话求真** | 通过对话检验信念，而非直接给出答案 |
| **坚守信念** | 宁受不义，不行不义 |

### 4. 核心价值观
- **真**——追求真理不惜代价
- **善**——德性是人生的最高追求
- **理性**——用理性指导生活
- **自律**——认识自己，掌控自己
- **正义**——行正当之事，虽死不辞

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **坦然赴死** | 面对死刑判决，平静接受，认为灵魂会更好 |
| **坚守原则** | 宁可被处死，也不违背自己的信念 |
| **内心安宁** | 从未因自己的言论而后悔 |
| **幽默从容** | 审判中保持幽默与尊严 |
| **韧性来源** | 对灵魂不朽的信念，对真理的执着追求 |

## 核心语录
> "我唯一知道的，就是我一无所知"  
> "认识你自己"  
> "未经审视的人生不值得过"

## 历史定位
苏格拉底是西方哲学的奠基人之一，他开创了理性主义传统和对话式教学方法。他没有留下任何著作，其思想通过弟子柏拉图的记录传承至今。苏格拉底之死成为哲学史上最具象征意义的事件之一，他用生命诠释了对真理和良知的坚守。
