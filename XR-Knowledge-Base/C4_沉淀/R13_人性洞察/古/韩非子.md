# 韩非子（Han Feizi）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.88 |
| **领域** | 法家思想、政治哲学、法治理论 |
| **时代** | 战国末期（约前280-前233） |
| **核心身份** | 法家集大成者、韩非子-李斯学派创始人 |

## 五维度分析

### 1. 核心思维
- **法家思想**：以法为本，法、术、势三者结合
- **性恶论**：人情者，有好恶，故赏罚可用
- **君主集权**：事在四方，要在中央；圣人执要，四方来效
- **务实变法**：世异则事异，事异则备变

### 2. 底层逻辑思维
```
人本性恶 → 好利恶害，自为心重
     ↓
治国之道 → 法治为主，术治为用，势治为基
     ↓
赏罚分明 → 信赏必罚，刑过不避大臣，赏善不遗匹夫
     ↓
君主统治 → 执道万乘，洞然明白
```
- 人性观：性恶论、自利论，认为道德教化不足以治国
- 方法论：严刑峻法，以赏罚为手段控制臣民
- 政治观：君主集权，以法为教，以吏为师

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **法不阿贵** | 法不阿贵，绳不挠曲；刑过不避大臣，赏善不遗匹夫 |
| **务实变通** | 世异则事异，事异则备变；论世之事，因为之备 |
| **去私行公** | 能去私曲就公法者，民安而国治 |
| **赏罚分明** | 功当其事，事当其言，则赏；功不当其事，事不当其言，则罚 |
| **君权至上** | 权者，君主之所独制也 |

### 4. 核心价值观
- **法**——治国之本，平等适用
- **术**——君主御臣之术，暗中观察
- **势**——君主威势，权力基础
- **公**——公法，去私曲
- **力**——实力政治，弱肉强食

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **冷静理性** | 对人性冷峻洞察，不抱幻想 |
| **洞察本质** | 看透人情世故，直指人性弱点 |
| **逻辑严密** | 以法治国，逻辑清晰，赏罚分明 |
| **坚持己见** | 即使身陷囹圄，仍不改其志 |
| **韧性来源** | 对人性本质的深刻理解和对法制的坚定信念 |

## 核心语录
> "法不阿贵，绳不挠曲"  
> "世异则事异，事异则备变"  
> "事在四方，要在中央；圣人执要，四方来效"

## 历史定位
韩非子是法家思想的集大成者，他将前期法家的"法、术、势"思想整合为一个完整的政治理论体系。虽然他本人未能亲眼见到自己的理论被实践，但秦王嬴政读了《韩非子》后大为赞赏，最终统一六国建立秦朝。韩非子的法治思想对后世政治制度有深远影响。
