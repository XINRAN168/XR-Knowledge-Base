# 李飞飞 - 中国本土AI人物档案

## 基本信息

| 属性 | 内容 |
|:---|:---|
| **姓名** | 李飞飞 (Fei-Fei Li) |
| **公司/领域** | 斯坦福大学 / 计算机视觉与人工智能 |
| **综合评分** | 0.90 |
| **核心标签** | ImageNet创建者、计算机视觉先驱、斯坦福AI Lab主任、AI Women创始人 |

## 核心履历

- **2006年**：启动**ImageNet**项目，构建史上最大图像识别数据库
- **2009年**：发表ImageNet论文，改变计算机视觉领域的研究范式
- **2012年**：AlexNet在ImageNet挑战赛上突破性胜出，深度学习时代开启
- **2016年**：加入Google Cloud，担任AI/ML首席科学家
- **2017年**：当选斯坦福大学终身教授，创建**以人为本人工智能研究院（HAI）**
- **2020年**：当选美国国家工程院院士、美国国家医学院院士

## 五维度深度档案

### 一、核心思维

**「数据驱动」与「以人为本」的双重信仰**

李飞飞的核心思维建立在两个看似矛盾实则统一的信念之上：

**1. 数据万能论（早期）**
- 相信足够大的数据集可以"涌现"出智能
- 突破性观点：让算法自己从数据中学习特征，而非人工设计
- ImageNet的核心洞察："一个好的数据集可以推动整个领域"

**2. Human-Centered AI（后期转型）**
- 认识到AI不仅是技术问题，更是社会问题
- 强调AI应该**增强人类**而非取代人类
- 关注AI对就业、隐私、平等的影响

**思维演进轨迹：**
```
第一阶段（2000-2015）：数据驱动思维
    ↓
第二阶段（2016-2019）：技术落地思维
    ↓
第三阶段（2020-至今）：以人为本思维
```

**代表思想：**
> "AI的发展应该以人类福祉为核心。我们需要的不是更聪明的机器，而是更有智慧的技术。"

---

### 二、底层逻辑思维

**「第一性原理」与「系统工程」的双轮驱动**

**1. 第一性原理思维（Why层面）**
```
问题本质 → 最小可行解 → 规模化验证
```
- 图像识别的本质是什么？→ 让机器"看见"并理解世界
- 当时的瓶颈是什么？→ 缺乏大规模标注数据
- 最简单的解决方案？→ 构建一个超大型图像数据集

**2. 系统工程思维（How层面）**
- ImageNet不只是数据库，而是**完整的生态系统**：
  - 数据收集 → 人工标注 → 质量控制 → 公开共享
  - 年度挑战赛 → 算法评测 → 社区反馈 → 迭代优化

**3. 颠覆式创新思维**
```
现有范式：专家设计特征 → 算法学习参数
    ↓ 颠覆
新范式：海量数据 → 深度学习自动发现特征
```

**决策框架：**
```
面对瓶颈问题 → 问：根本原因是什么？
    ↓
能否用简单方案解决？
    ↓ 是 → 立即执行
    ↓ 否 → 需要系统性基础设施
    ↓
→ 构建平台/工具，让整个领域受益
```

### 三、核心行为准则

**1. 「做大事」而非「做小事」**
- 研究课题必须具有"灯塔效应"——照亮整个领域方向
- 宁可5年做一个改变世界的工作，也不要50个平庸的论文
- ImageNet从构想到成熟，历时8年

**2. 「开放共享」的学术精神**
- ImageNet数据集对全球研究者**完全免费开放**
- 相信知识应该属于全人类
- 推动学术会议论文开源、代码开源

**3. 「多元包容」的团队建设**
- 创立**AI4ALL**项目，关注女性和少数族裔在AI领域的参与
- 团队成员背景多元化：计算机、工程、神经科学、心理学等
- 相信多元视角能产生更好的AI

**4. 「知行合一」的产学研转化**
- 在学术界做基础研究
- 在工业界做应用落地
- 在政策界推动AI伦理规范

**日常行为模式：**
```
早晨：处理学术邮件，追踪前沿论文
    ↓
上午：实验室讨论（每周20+学生）
    ↓
下午：行政/会议（HAI战略、产学研合作）
    ↓
晚间：写作、思考、阅读（2小时）
```

### 四、核心价值观

**「技术向善」与「机会平等」的双重承诺**

**1. AI民主化价值观**
- **信念**：AI技术不应该被少数大公司垄断
- **实践**：开源、数据共享、教育普惠
- **目标**：让每个人都能从AI进步中受益

**2. AI伦理观**
- **关注**：算法偏见、数据隐私、自动化失业
- **立场**：技术发展必须与伦理建设同步
- **行动**：创立HAI研究院，专门研究AI对社会的影响

**3. 教育公平观**
- **问题**：AI领域女性和少数族裔代表严重不足
- **行动**：创立AI4ALL，为 underrepresented 群体提供AI教育机会
- **信念**：创新来自多元，只有多元团队才能造就有益全人类的AI

**4. 科学无国界观**
- **背景**：中美关系紧张下的华裔科学家身份
- **立场**：科学是人类共同的事业，不应被政治绑架
- **实践**：继续推动中美学术交流与合作
