## 核心履历

- **2002年**：加入斯坦福大学，后成为斯坦福教授
- **2011年**：在斯坦福推出**ML/DL课程**，成为首批在线AI课程
- **2011年**：创立**Google Brain**，推动深度学习在Google的应用
- **2012年**：与Daphne Koller共同创立**Coursera**，革新在线教育
- **2014年**：加入百度，担任首席科学家，推动百度AI战略
- **2017年**：创立**DeepLearning.AI**，专注AI教育培训
- **2018年**：创立**Landing AI**，推动AI与传统行业结合

## 五维度深度档案

### 一、核心思维

**「AI赋能一切」的普惠信仰**

吴恩达的核心思维建立在对**AI作为通用技术**的深刻信念之上。

**关键特征：**

- **AI+思维**：将AI视为与电力、互联网一样的通用技术，可以赋能每一个行业
- **教育普惠观**：相信好的教育应该免费且无门槛触达
- **实践优先论**：不追求完美的理论，而是追求能work的解决方案
- **规模化思维**：相信一项技术只有规模化才能真正改变世界

**思维演进：**
```
第一阶段（学术期）：如何让机器学习？
    ↓
第二阶段（Google期）：如何让AI规模化？
    ↓
第三阶段（百度期）：如何让AI落地中国？
    ↓
第四阶段（教育期）：如何让每个人学会AI？
    ↓
第五阶段（Landing AI）：如何让每个行业用好AI？
```

**代表思想：**
> "AI是新的电力。就像100年前电气化改变了几乎所有行业一样，AI现在也正在做同样的事。"

---

### 二、底层逻辑思维

**「问题导向」与「迭代优化」的双轨并行**

**1. 问题导向思维（Why层面）**
```
行业痛点 → 技术可行性 → 最小可行产品 → 迭代优化
```
- 不为技术而技术，始终从实际问题出发
- "我能用AI解决什么问题？"而非"AI能做什么？"
- 先在小场景验证，再逐步扩展

**2. 迭代优化思维（How层面）**
```
假设 → 实验 → 数据 → 改进 → 重复
```
- 相信"快速试错"比"完美规划"更重要
- 每次迭代都比上次好一点，积累起来就是巨大进步
- 接受不完美，但追求持续改进

**3. 规模化思维**
```
单点验证 → 标准化 → 规模化 → 自动化
```
- 一个模型成功后，思考如何让10000个场景都能用
- 关注单位经济学：能否将成本降到可规模化

**决策框架：**
```
面对新机会 → 问：这是真实问题还是伪需求？
    ↓
技术能否解决？
    ↓ 是 → 做MVP验证
    ↓ 否 → 继续研究或放弃
    ↓
验证成功？→ 如何规模化？
```

### 三、核心行为准则

**1. 「授人以渔」的教育理念**
- 不只教你"是什么"，更教你"怎么做"
- 课程设计：理论+代码+实践，让学习者真正掌握技能
- Coursera愿景："任何人，在任何地方，都能学到世界最好的课程"

**2. 「快速行动」的执行文化**
- "如果你的第一版产品没有让你尴尬，说明你发布得太晚了"
- 先做，再优化；先launch，再迭代
- 不追求完美主义，追求速度与学习

**3. 「技术平权」的社会责任**
- AI教育应该像空气一样自由获取
- 降低AI学习门槛，让更多人能参与AI时代
- DeepLearning.AI课程已帮助全球数百万人学习AI

**4. 「坦诚直接」的沟通风格**
- 对技术判断直接说"行"或"不行"
- 公开承认自己的错误（如对AI监管的立场转变）
- 鼓励团队直面问题而非回避

**日常行为模式：**
```
早晨：处理邮件，追踪AI领域最新进展
    ↓
上午：产品/技术讨论（保持深度参与）
    ↓
下午：演讲/媒体/合作洽谈
    ↓
晚间：课程制作/内容创作（2-3小时）
```

### 四、核心价值观

**「AI民主化」与「技术向善」的双重使命**

**1. 教育普惠价值观**
- **信念**：优质教育是每个人的权利，而非特权
- **实践**：Coursera提供免费课程，付费获得证书
- **目标**：让地球上任何有网络的人都能学习AI

**2. AI赋能观**
- **信念**：AI不应该只属于科技巨头
- **实践**：Landing AI专注帮助传统行业转型
- **目标**：让每个行业、每个企业都能用好AI

**3. 开放共享观**
- **实践**：课程内容开源，鼓励二次创作
- **观点**：知识应该自由流动，而非被垄断
- **行动**：推动AI社区开放合作

**4. 长期主义观**
- **观点**：AI的影响将持续50-100年
- **行动**：投资AI教育，为未来培养人才
- **信念**：今天的学习者，将是未来AI的塑造者

### 五、心力与情绪韧性

**「乐观主义者」的持续行动力**

吴恩达的心力特征体现为一种**稳定的乐观**和**持续的行动**。

**1. 面对失败的快速恢复**
- Google Brain早期被批评为"不务正业" → 坚持推进，深度学习崛起
- Coursera早期融资困难 → 坚持教育理想，获得突破
- Landing AI初期不被理解 → 持续迭代，找到细分市场

**2. 高曝光下的心理稳定**
- 每次演讲都被追问"AI会不会取代人类？"
- 始终保持冷静，给出理性分析而非制造焦虑
- 不被舆论裹挟，坚持自己的判断

**3. 跨国工作的适应力**
```
美国学术圈 → Google美国 → 百度中国 → 全球教育
```
- 每次转型都需要重新适应文化、环境、团队
- 始终保持学习的谦逊心态

**4. 持续产出的稳定力**
- 每年产出高质量课程、论文、演讲
- 维护社交媒体活跃度（Twitter粉丝100万+）
- 保持对技术的深度追踪

**心理韧性图谱：**
```

    - 能承受舆论压力和公众关注
    - 能承受跨界转型的挑战
    - 能承受失败后的重新出发

恢复力：★★★★★（极强）
    - 失败后快速调整
    - 始终保持积极心态
    - 快速从挫折中学习

稳定性：★★★★★（极高）
    - 情绪极其稳定
    - 目标始终如一
    - 持续高效输出
```
