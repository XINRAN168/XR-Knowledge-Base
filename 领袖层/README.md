## 🧠 档案标准格式

每个领袖档案包含以下维度：

1. **核心思维** - 领袖的决策逻辑和思考方式
2. **底层逻辑思维** - 形成核心思维的根源
3. **核心行为准则** - 贯穿始终的行事风格
4. **核心价值观** - 治国理念和价值取向
5. **心力/情绪韧性** - 抗压能力和情绪管理
6. **综合评价** - 多维度评分与学习建议
7. **关键决策案例** - 重大历史决策分析
