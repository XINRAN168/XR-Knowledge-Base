# 华佗 - 外科圣手

> **评分：0.88** | 时代：东汉 | 核心标签：麻沸散、五禽戏、外科鼻祖

## 一、核心思维

### 1.1 外科手术思维
- **核心理念**：以手术解决药物难以奏效的病症
- **核心表现**：
  - 麻沸散全身麻醉
  - 剖腹、开颅等大型手术
  - 术后康复管理

### 1.2 预防养生思维
- **核心理念**："治未病"重于"治已病"
- **核心表现**：
  - 创编五禽戏
  - 强调体育锻炼
  - "人体欲得劳动，但不当使极耳"

### 1.3 实证疗效思维
- **核心理念**：以临床疗效为检验标准
- **核心表现**：
  - 精简药方，精准用药
  - 针灸取穴精简高效
  - 手术操作规范流程

## 二、底层逻辑思维

### 2.1 对症施治
- **运作方式**：根据病情性质选择最优治疗路径
- **典型案例**：陈登胃中虫积，以吐下法治愈

### 2.2 内外结合
- **运作方式**：手术与药物、针灸综合运用
- **典型案例**：刮骨疗毒配合术后康复

### 2.3 动静结合
- **运作方式**：运动养生与药物治疗结合
- **典型案例**：五禽戏配合药饵调理

## 三、核心行为准则

### 3.1 诊疗准则
| 准则 | 内容 |
|:---|:---|
| 诊断精准 | 脉诊准确，判断无误 |
| 用药精简 | "合汤不过数种"，心解分剂 |
| 针灸简捷 | 取穴不多，针到病除 |
| 手术规范 | 术前麻醉，术中无菌，术后康复 |

### 3.2 手术准则
| 步骤 | 操作 |
|:---|:---|
| 麻醉 | 麻沸散酒服，失去知觉 |
| 切口 | 精准切开，避开要害 |
| 探查 | 查明病变，清洗处理 |
| 缝合 | 桑皮线缝合，敷以神膏 |
| 康复 | 四五日除痛，一月痊愈 |

### 3.3 养生准则
> "户枢不朽"，生命在于运动

- 模仿五禽：虎、鹿、熊、猿、鸟
- 因人而异：针对不同体质设计
- 持之以恒：长期锻炼方见成效
