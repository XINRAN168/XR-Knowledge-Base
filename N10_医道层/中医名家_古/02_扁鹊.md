# 扁鹊 - 四诊法奠基人

> **评分：0.88** | 时代：战国 | 核心标签：望闻问切、四诊法

## 一、核心思维

### 1.1 四诊合参诊断观
- **核心理念**：综合运用望、闻、问、切四种诊法，避免单一诊断偏差
- **核心表现**：
  - 望诊：观察面色、神态、形体、舌苔
  - 闻诊：听声音、嗅气味
  - 问诊：详细询问病史、症状
  - 切诊：寸口脉诊，独取寸口

### 1.2 阴阳辨证思维
- **核心理念**：以阴阳解释病因，疾病源于阴阳失调
- **核心表现**：
  - "闻病之阳，论得其阴；闻病之阴，论得其阳"
  - 调节机体阴阳平衡是治疗关键
  - 病邪沿经络传变，由表入里

### 1.3 整体联系思维
- **核心理念**：脏腑与经络相互联络，疾病可循经传变
- **核心表现**：
  - 脏腑经络结合分析病因病机
  - 病邪沿经络由表入里传变

## 二、底层逻辑思维

### 2.1 司外揣内
- **运作方式**：通过外在表现推断内在病因
- **典型案例**：望齐桓公面色知病情深浅

### 2.2 循经辨治
- **运作方式**：正确循经取穴进行针刺治疗
- **典型案例**：虢太子尸厥诊治

### 2.3 实证诊断
- **运作方式**：以客观诊查为依据，排除主观臆断
- **典型案例**：寸口诊法确立

## 三、核心行为准则

### 3.1 诊断准则
| 准则 | 内容 |
|:---|:---|
| 四诊互参 | 四法综合，不可偏废 |
| 循经辨治 | 按经络循行取穴 |
| 早期发现 | 病在腠理即治，防微杜渐 |

### 3.2 治疗准则
| 准则 | 内容 |
|:---|:---|
| 因证施治 | 根据辨证结果对症下药 |
| 针药并用 | 砭灸、方药综合治疗 |
| 补虚泻实 | 调整虚实，恢复平衡 |

### 3.3 医德准则
> "信巫不信医者不治"

- 反对鬼神迷信
- 提倡科学实证
- 普同一等救治
