# 伽利略 Galileo Galilei (1564-1642)

**评分：0.88**

**科学贡献：**
- 自由落体定律
- 惯性定律雏形
- 摆的等时性
- 望远镜天文观测
- 科学方法论奠基

---

## 一、核心思维

**数学化自然**
- "自然之书是用数学语言写成的"
- 用精确数学公式描述物理世界
- 物理规律可量化为数学关系

**实验-数学结合**
- 不是被动观察，而是主动设计实验
- 用可控实验"拷问自然"
- 隔离变量，研究因果关系

**理想化抽象**
- 引入"理想实验"概念
- 在思维中创造没有干扰的理想环境
- 导出纯粹物理定律

**质疑权威**
- 用逻辑驳斥亚里士多德两千年权威
- "自然知识不应来自对古代权威的盲从"
- 从自然本身探究真理

---

## 二、底层逻辑思维

**归谬法**
```
假设A → 逻辑推导 → 得出矛盾 → 否定A
```

**实验归纳法**
```
观察现象 → 提出假说 → 实验验证 → 修正假说 → 形成理论
```

**数学外推法**
- 斜面实验 → 自由落体
- 从可测量的现象外推到极限情况

**变量控制**
- 隔离单个变量
- 系统性改变条件
- 记录因果关系

---

## 三、核心行为准则

1. **眼见为实**
   - 亲手制造望远镜观测天体
   - 用观测证据挑战地心说

2. **量化思维**
   - 距离与时间的平方成正比（h∝t²）
   - 用数字而非形容词描述

3. **大胆假设**
   - 提出革命性假设（惯性、相对性）
   - 不惧与传统对立

4. **谨慎求证**
   - 假设需实验验证
   - 不轻易下结论

5. **为真理斗争**
   - 面对宗教裁判所
   - 坚持科学真理

## 四、核心价值观

| 维度 | 内涵 |
|:---:|:---|
| **真理观** | 真理来自自然观察，非权威 |
| **方法观** | 实验+数学+逻辑是认识自然的正道 |
| **权威观** | 挑战权威，但不盲从 |
| **科学观** | 科学应描述"如何"而非追问"为何" |
