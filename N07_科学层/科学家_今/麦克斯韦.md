# 麦克斯韦 James Clerk Maxwell (1831-1879)

**评分：0.88**

**科学贡献：**
- 麦克斯韦方程组（经典电动力学基础）
- 电磁波预言（光=电磁波）
- 麦克斯韦-玻尔兹曼分布
- 气体动理论
- 卡文迪什实验室创建者

---

## 一、核心思维

**数学形式美**
- 将法拉第的直觉翻译成数学方程
- 追求方程的对称性与简洁性
- "数学即物理"的信仰

**统一思维**
- 统一电、磁、光于同一框架
- 用四个方程描述所有电磁现象
- 预言自然界最深刻的统一

**场论思维**
- 拒绝超距作用
- 提出"场"作为物理实体
- 比物理学主流早40年

**数学预言**
- 引入"位移电流"概念（纯数学项）
- 预言电磁波存在
- 用数学发现物理实在

---

## 二、底层逻辑思维

**公理化思维**
```
法拉第直觉 → 数学翻译 → 方程构建 → 推论导出 → 实验验证
```

**数学一致性驱动**
- 从方程自洽性发现新项（位移电流）
- 数学要求成为物理发现

**极限思维**
- 从特殊到一般（气体分子→统计描述）
- 揭示隐藏在现象背后的结构

**拓扑思维**
- 将力线看作物理实在的拓扑结构
- 空间本身具有可计算的力学属性

---

## 三、核心行为准则

1. **数学严谨**
   - 方程是骨架，物理是血肉
   - 追求逻辑自洽

2. **理论优先**
   - 在实验验证前相信数学预言
   - 方程预言必真

3. **拒绝直观束缚**
   - 删去所有以太力学假设
   - "场即实体，无需载体"

4. **简洁追求**
   - 20个方程压缩为4个
   - 强调协变性，预示相对论

5. **承前启后**
   - 系统概括法拉第思想
   - 为后人铺路
