- 《物种起源》

- 人类起源理论

## 一、核心思维

**归纳观察法**
- 二十年的观察与资料收集
- 从大量实例中归纳规律
- "物种不是被创造的，而是演化的"

**系统比较法**
- 比较不同物种的形态结构
- 比较地理分布
- 比较肧胎发育

**证据积累**
- 地质学、古生物学、比较解剖学证据
- 遗传变异的观察
- 人工选择的观察

**自然选择机制**
- 过度繁殖与生存斗争
- 变异的存在
- 适者生存

## 二、底层逻辑思维

**归纳法**
```
大量观察 → 归纳规律 → 形成理论 → 更多验证
```

**因果推理**
- 变异 → 遗传 → 筛选 → 适应
- 结果反推原因

**概率思维**
- 承认变异的随机性
- 自然选择的方向性
- 累积效应的长期性
