# 钱学森 Qian Xuesen (1911-2009)

**评分：0.92**

**科学贡献：**
- 工程控制论（奠基人）
- 中国航天之父
- 中国导弹之父
- 系统科学思想
- 两弹一星功勋

---

## 一、核心思维

**系统论思维**
- 将工程问题视为动态系统
- 强调整体大于部分之和
- 从定性到定量综合集成

**工程科学观**
- 连接基础科学与工程实践的桥梁
- 理论必须可操作
- 追求"从定性到定量"的转化

**跨学科整合**
- 融合数学、物理学、工程学、信息论
- 打通不同学科边界
- 综合集成解决问题

**反馈思维**
- 控制论核心：反馈机制
- 系统自我修正
- 稳定性与可控性

---

## 二、底层逻辑思维

**从定性到定量**
```
实际问题 → 定性认识 → 数学建模 → 定量分析 → 工程实现
```

**系统分解-整合**
```
复杂系统 → 子系统分解 → 分层优化 → 系统整合
```

**理论-实践循环**
```
理论 → 实践检验 → 反馈修正 → 理论完善
```

**工程化思维**
- 建模→仿真→实验→验证
- 可行性分析
- 约束条件处理

## 三、核心行为准则

1. **工程导向**
   - 理论必须可应用于工程
   - "工程控制论不只是理论，更是实践"

2. **系统观点**
   - 从全局看问题
   - 强调整体优化

3. **实践验证**
   - 理论必须在实践中检验
   - 在两弹一星工程中实践理论

4. **人才培养**
   - 创建中国航天体系
   - 培养大批科技人才

5. **科学报国**
   - 冲破阻力回国
   - 以科技服务国家
