# 戚继光 - 抗倭名将

## 五维度档案

### 1. 核心思维
- **兵者因地制宜**：根据实际地形和敌人特点设计战术
- **鸳鸯阵法**：阵法创新，以小团队协同作战
- **选兵为先**：精兵路线，士兵素质比数量更重要
- **纪律严明**：令行禁止，赏罚分明

### 2. 底层逻辑思维
```
敌情分析 → 武器装备适配 → 阵法设计 → 训练体系 → 实战检验 → 迭代优化
```
- 系统工程：从选兵、训练、装备、战术形成完整体系
- 持续改进：战后必总结，不断优化战术
- 实战导向：一切训练以实战为目的

### 3. 核心行为准则
- **有功必赏**：赏赐及时到位，激励士气
- **有罪必罚**：纪律严明，不姑息迁就
- **与士兵同甘苦**：将领与士兵同吃同住同战斗
- **以身作则**：将领必须冲在最前面

### 4. 核心价值观
- **保家卫国**：抵御外侮，保护百姓
- **革新图强**：不拘泥传统，勇于改革创新
- **知行合一**：理论与实践相结合
- **传承精神**：著书立说，将军事思想系统化

### 5. 心力/情绪韧性
- **百折不挠**：面对困难和挫折不退缩
- **逆境成长**：从失败中吸取教训，不断成长
- **隐忍坚持**：面对朝堂阻力，依然坚持改革
- **矢志不渝**：一生只为一件事——抗倭报国
