# 霍去病 - 封狼居胥

## 五维度档案

### 1. 核心思维
- **闪电战**：快速突袭，不给敌人反应时间
- **长途奔袭**：跨越千里，直捣黄龙
- **斩首行动**：以消灭敌军指挥系统为核心目标
- **骑兵为王**：充分发挥骑兵机动优势

### 2. 底层逻辑思维
```
情报先行 → 快速集结 → 闪电突袭 → 精准打击 → 即时撤退
```
- 速度制胜：在敌人反应过来之前结束战斗
- 轻装上阵：不带辎重，就地取食
- 心理震慑：以高速机动让敌人产生恐惧

### 3. 核心行为准则
- **兵贵神速**：行军速度决定战争胜负
- **果敢决断**：战场机会稍纵即逝，必须果断行动
- **不拘古法**：不按兵书套路出牌
- **锐意进取**：永远向前，永不退缩

### 4. 核心价值观
- **开疆拓土**：为汉朝开疆拓土是最高荣誉
- **匈奴未灭何以为家**：国家利益高于个人家庭
- **少年英雄**：年轻不代表能力不足
- **一战封神**：用最辉煌的战绩证明自己

### 5. 心力/情绪韧性
- **少年老成**：年纪轻轻却有钢铁意志
- **无畏无惧**：面对强敌毫无惧色
- **意气风发**：永远保持高昂斗志
- **天妒英才**：虽英年早逝，但精神永存
