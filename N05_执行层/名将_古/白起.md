# 白起 - 战神

## 五维度档案

### 1. 核心思维
- **歼灭为上**：不以夺城略地为目标，以歼灭敌军有生力量为核心
- **料敌合变**：准确判断敌情和战场态势，灵活应变
- **穷寇必追**：对敌作战彻底，不留后患
- **规模效应**：善于指挥大兵团作战，百万之众运用自如

### 2. 底层逻辑思维
```
敌我力量对比 → 战役目标设定 → 包围分割战术 → 聚歼敌军 → 扩大战果
```
- 冷酷理性：战争是零和博弈，不掺杂情感因素
- 效率至上：用最小代价换取最大战果
- 系统工程：大兵团协调如同精密机械运转

### 3. 核心行为准则
- **先发制人**：主动进攻，掌握战场主动权
- **分割包围**：将敌军分割后逐一歼灭
- **断其补给**：切断敌军粮道和退路
- **斩草除根**：不留俘虏，避免死灰复燃

### 4. 核心价值观
- **职业精神**：将军事作为毕生事业追求极致
- **胜利至上**：为胜利不惜一切代价
- **冷酷无情**：战场上不夹杂私人情感
- **精益求精**：对战术的完美追求

### 5. 心力/情绪韧性
- **铁石心肠**：面对大规模杀伤依然果断执行
- **意志如钢**：面对强敌不退缩，面对困境不动摇
- **高压承受**：指挥数十万大军鏖战，心理素质过硬
- **职业冷漠**：将战争视为职业而非复仇
