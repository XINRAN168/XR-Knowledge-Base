# 拿破仑 - 军事天才

## 五维度档案

### 1. 核心思维
- **集中优势兵力**：在决定性地点集中绝对优势兵力
- **机动作战**：快速机动，切割包围敌人
- **拿破仑方阵**：军事技术创新，以纵队与散兵结合
- **政治军事结合**：军事服务于政治目标

### 2. 底层逻辑思维
```
战略布局 → 情报分析 → 快速机动 → 分割包围 → 决定性打击 → 政治整合
```
- 系统思维：将战争视为系统工程
- 持续创新：不断改进战术和战略
- 效率优先：用最小代价获取最大胜利

### 3. 核心行为准则
- **速度制胜**：行军速度决定战争胜负
- **果断决策**：战场机会稍纵即逝
- **亲自指挥**：将领必须在战场最前线
- **激励士气**：善于用演说激励军队

### 4. 核心价值观
- **法兰西至上**：为法兰西的荣耀而战
- **革命理想**：传播法国大革命理念
- **精英统治**：相信能力和功绩决定地位
- **法律秩序**：用法律和秩序治理国家

### 5. 心力/情绪韧性
- **逆境崛起**：从科西嘉小个子到法兰西皇帝
- **永不言败**：即使流放也要东山再起
- **百折不挠**：失败后能迅速调整
- **命运抗争**：滑铁卢之前始终相信自己
