# 关羽 - 武圣

## 五维度档案

### 1. 核心思维
- **忠义无双**：忠君爱国、义薄云天是立身之本
- **单刀赴会**：勇气与胆识并重，敢于孤身犯险
- **水淹七军**：利用自然力量，创造战场优势
- **威震华夏**：以赫赫威名让敌人闻风丧胆

### 2. 底层逻辑思维
```
敌情判断 → 勇气驱动 → 精准打击 → 威名传播
```
- 义气为先：在利益与义气之间，永远选择义气
- 正面交锋：不屑于阴谋诡计，正面击败敌人
- 威名效应：用赫赫威名不战而屈人之兵

### 3. 核心行为准则
- **忠义不移**：身在曹营心在汉，不负刘备知遇之恩
- **恩怨分明**：有恩必报，有仇必报
- **光明磊落**：行事正大光明，不搞阴谋诡计
- **义释曹操**：为报旧日恩情，放过主要敌人

### 4. 核心价值观
- **忠义为本**：忠君爱国是最高准则
- **义薄云天**：为朋友可以两肋插刀
- **光明正大**：行事坦荡，不搞小动作
- **知恩图报**：受人滴水之恩，必当涌泉相报

### 5. 心力/情绪韧性
- **刮骨疗毒**：能忍常人所不能忍之痛
- **威武不屈**：面对强敌绝不低头
- **傲骨铮铮**：骄傲但不傲慢，永不屈服
- **慷慨就义**：即使失败，也要死得壮烈
