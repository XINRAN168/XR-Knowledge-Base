# 汉尼拔 - 迦太基名将

## 五维度档案

### 1. 核心思维
- **越险而出**：率军跨越阿尔卑斯山，出其不意
- **以少胜多**：坎尼之战，以弱胜强的典范
- **后勤独立**：不依赖后方补给，就地解决
- **心理战**：通过胜利建立军队信心

### 2. 底层逻辑思维
```
战略奇袭 → 孤军深入 → 就地补给 → 连续作战 → 瓦解敌方
```
- 孤注一掷：将全国之力押注于一场战争
- 敌后作战：深入敌境，远离己方支援
- 心理优势：用胜利建立不可战胜的威名

### 3. 核心行为准则
- **冒险精神**：敢于做别人不敢做的事
- **独立作战**：不依赖援军，靠自己取胜
- **灵活机动**：敌变我变，永远掌握主动权
- **身先士卒**：将领与士兵同甘共苦

### 4. 核心价值观
- **国家使命**：为迦太基效忠是最高荣誉
- **军事至上**：相信军事手段能解决一切
- **个人荣耀**：追求作为军事指挥官的荣耀
- **知恩图报**：对赏识自己的人感恩戴德

### 5. 心力/情绪韧性
- **越战越勇**：从失败中崛起，永不放弃
- **逆境坚守**：孤军在敌境坚持十五年
- **背水一战**：没有退路，只能向前
- **视死如归**：为国家献出一切
