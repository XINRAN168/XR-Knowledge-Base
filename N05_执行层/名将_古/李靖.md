# 李靖 - 唐朝军神

## 五维度档案

### 1. 核心思维
- **攻守兼备**：不仅善于进攻，更精于防守
- **料敌先机**：通过情报分析预判敌军动向，先发制人
- **因粮于敌**：善于利用敌方资源补给己方
- **虚实结合**：真虚实实，让敌人无法判断真实意图

### 2. 底层逻辑思维
```
情报收集 → 敌情研判 → 战略规划 → 战术欺骗 → 精准打击 → 战后安抚
```
- 风险控制：每战必先想好退路
- 全局统筹：不仅考虑战役本身，更考虑政治影响
- 和平方针：战争最终目的是和平

### 3. 核心行为准则
- **稳扎稳打**：不贪功冒进，每一步都有把握
- **见机行事**：战场形势瞬息万变，善于把握战机
- **仁义之师**：作战同时安抚百姓，不滥杀无辜
- **知进知退**：知道何时该进，何时该退

### 4. 核心价值观
- **忠君报国**：以国家利益为重，不计较个人得失
- **仁者无敌**：真正的强者不需要依靠杀戮证明
- **低调务实**：功成不居，不争名夺利
- **传承精神**：著书立说，将军事思想传给后人

### 5. 心力/情绪韧性
- **宠辱不惊**：面对赞扬和批评都能保持平常心
- **临阵沉静**：千军万马中依然能冷静判断
- **以柔克刚**：不逞一时之勇，追求最终胜利
- **淡泊明志**：不慕名利，专心军事
