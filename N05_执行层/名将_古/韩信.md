# 韩信 - 兵仙

## 五维度档案

### 1. 核心思维
- **多多益善**：大规模兵团指挥能力，对兵力运用有极致理解
- **出奇制胜**：不按常理出牌，以非常规战术创造战机
- **背水一战**：置之死地而后生，激发部队最大战斗意志
- **破楚灭齐**：战略布局清晰，先弱后强各个击破

### 2. 底层逻辑思维
```
敌情研判 → 战术创新 → 地形利用 → 士气激发 → 精准执行
```
- 反直觉思维：常人不屑的绝境正是其创造奇迹的舞台
- 心理学应用：深谙人性弱点，善于利用对手心理
- 模块化思维：将复杂战役分解为可执行的简单动作

### 3. 核心行为准则
- **攻心为上**：心理战与实际作战双管齐下
- **以少胜多**：不追求绝对力量优势，追求战术优势
- **因地制宜**：充分利用地形、地势创造有利态势
- **果敢决断**：战机稍纵即逝，敢于冒险抓住机会

### 4. 核心价值观
- **专业至上**：对军事艺术的极致追求
- **结果导向**：无论手段，只看最终胜利
- **知恩图报**：功成名就后不忘初心
- **功成身退**：深谙鸟尽弓藏之理

### 5. 心力/情绪韧性
- **胯下之辱能忍**：能承受常人不能忍的屈辱
- **逆境自若**：从贫困潦倒到封侯拜将，内心始终强大
- **临阵从容**：千军万马中依然保持冷静判断
- **成败不惊**：无论胜败都能快速调整心态
