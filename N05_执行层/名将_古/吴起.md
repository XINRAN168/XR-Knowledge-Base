# 吴起 - 兵法大家

## 五维度档案

### 1. 核心思维
- **内修文德**：重视军队思想建设，与严明纪律并重
- **外治武备**：加强军事实力，精进战术训练
- **因势利导**：根据形势变化灵活调整战略战术
- **以治待乱**：以严明纪律和稳定阵型应对混乱战场

### 2. 底层逻辑思维
```
政治与军事融合 → 制度体系建设 → 训练体系打造 → 激励与约束并重
```
- 综合国力观：战争胜负取决于综合国力而非单纯军事力量
- 制度化管理：用制度而非个人魅力管理军队
- 持续改进：战后必复盘，不断优化战术体系

### 3. 核心行为准则
- **与士卒同甘苦**：与最基层士兵同吃同住，赢得爱戴
- **令行禁止**：军令如山，违者必究
- **赏罚分明**：有功必赏，有过必罚，绝不姑息
- **因材施教**：根据士兵特点分配适合的岗位

### 4. 核心价值观
- **以民为本**：战争服务于国家安全和人民利益
- **德才兼备**：将领不仅要有才能，更要有品德
- **忠君爱国**：以国家利益为最高准则
- **改革创新**：不拘泥于传统，勇于改革军制

### 5. 心力/情绪韧性
- **能屈能伸**：在魏则留，去楚则往
- **逆境坚守**：无论环境如何恶劣，坚持自己的军事理念
- **批评承受**：面对非议和攻击不动摇
- **孤独前行**：改革者注定孤独，但依然坚持
