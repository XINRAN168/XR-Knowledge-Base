# 岳飞 - 精忠报国

## 五维度档案

### 1. 核心思维
- **文官不爱财、武官不惜死**：将领必须有牺牲精神和廉洁品质
- **运用之妙、存乎一心**：战术灵活运用，全凭内心感悟
- **连结河朔**：建立广泛的抗金统一战线
- **以少敌众**：精兵路线，以质量胜数量

### 2. 底层逻辑思维
```
政治动员 → 军队建设 → 战术创新 → 民众支持 → 北伐恢复
```
- 正义战争观：战争必须师出有名，以恢复故土为旗帜
- 民心向背：得民心者得天下，战争必须争取民众支持
- 精神力量：用信仰和荣誉激发部队战斗力

### 3. 核心行为准则
- **冻死不拆屋**：严格要求部队爱护百姓
- **饿死不掳掠**：保持仁义之师形象
- **身先士卒**：将领冲在最前面，与士卒同甘共苦
- **令行禁止**：岳家军纪律严明，号令统一

### 4. 核心价值观
- **精忠报国**：以报效国家为最高人生目标
- **民族大义**：抵御外侮，恢复中华
- **忠孝两全**：忠于国家，孝于民族
- **气节至上**：宁为玉碎，不为瓦全

### 5. 心力/情绪韧性
- **坚贞不屈**：被诬陷入狱仍不改其志
- **逆境坚守**：十二道金牌召不回北伐决心
- **家国情怀**：将个人命运与国家命运紧密结合
- **视死如归**：明知必死，仍不改精忠报国之心
