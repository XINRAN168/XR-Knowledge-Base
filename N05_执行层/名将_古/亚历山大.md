# 亚历山大大帝 - 马其顿征服者

## 五维度档案

### 1. 核心思维
- **马其顿方阵**：军事技术创新，以严密的步兵方阵为核心
- **快速机动**：以速度和机动性压制敌人
- **心理征服**：不仅占领领土，更要征服人心
- **文化融合**：尊重被征服地区的文化，实现共治

### 2. 底层逻辑思维
```
军事打击 → 领土占领 → 民心征服 → 文化融合 → 帝国巩固
```
- 扩张逻辑：军事征服是帝国发展的核心动力
- 精英培养：培养当地精英，建立傀儡政权
- 持续推进：不断向前，不给敌人喘息机会

### 3. 核心行为准则
- **身先士卒**：将领必须冲在最前面
- **冒险精神**：敢于以少敌多，创造奇迹
- **宽宏大量**：善待投降者，化敌为友
- **永不停歇**：永远向前，不满足于现有成就

### 4. 核心价值观
- **荣耀至上**：追求永恒的荣耀和名声
- **英雄主义**：崇拜英雄，追求成为最伟大的人
- **文明传播**：将希腊文明传播到世界各地
- **普世王权**：相信自己是世界的主宰

### 5. 心力/情绪韧性
- **无畏无惧**：面对任何困难和危险都毫无惧色
- **永不言败**：即使遭遇挫折也不放弃
- **自我超越**：永远追求超越自己
- **命运掌控**：相信自己掌控着自己的命运
