# 成吉思汗 - 蒙古帝国

## 五维度档案

### 1. 核心思维
- **闪电战**：快速突击，不给敌人反应时间
- **心理战**：通过屠杀制造恐惧，瓦解敌人士气
- **联盟战略**：善于建立广泛的军事联盟
- **情报网络**：建立覆盖欧亚的情报网络

### 2. 底层逻辑思维
```
统一蒙古 → 建立同盟 → 情报先行 → 闪电突袭 → 分化瓦解 → 统治整合
```
- 征服逻辑：征服是帝国发展的核心
- 以战养战：利用敌方资源补给己方
- 组织创新：千户制度，军事化管理

### 3. 核心行为准则
- **令行禁止**：绝对服从，铁的纪律
- **赏罚分明**：有功必赏，有过必罚
- **宽严并济**：对敌人残酷，对部下宽厚
- **因地制宜**：根据不同敌人采取不同策略

### 4. 核心价值观
- **征服荣耀**：征服是最高荣誉
- **忠诚至上**：对主君和部下的忠诚
- **实力说话**：能力和功绩决定地位
- **蒙古至上**：为蒙古民族的荣耀而战

### 5. 心力/情绪韧性
- **逆境崛起**：从部落孤儿到帝国皇帝
- **屡败屡战**：失败后能迅速重整旗鼓
- **意志如铁**：任何困难都不能阻止他
- **老当益壮**：六十岁依然率军西征
