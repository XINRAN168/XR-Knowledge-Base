# 孙武 - 兵圣

## 五维度档案

### 1. 核心思维
- **知己知彼**：一切战争决策建立在充分情报与敌我态势分析之上
- **庙算先胜**：战前在庙堂之上通过战略推演预判胜负概率
- **不战而屈人之兵**：最高军事艺术是以威慑而非实战达成目标
- **因敌变化而取胜**：灵活机动，根据战场实时变化调整战术

### 2. 底层逻辑思维
```
条件输入 → 态势分析 → 庙算推演 → 策略制定 → 灵活执行 → 复盘迭代
```
- 量化思维：用"道天地将法"五维框架系统评估战争条件
- 风险对冲：先为不可胜，以待敌之可胜
- 成本收益：杀敌一万自损三千的下策，追求以小博大

### 3. 核心行为准则
- **先计后战**：任何军事行动必须有完整的事前推演和预案
- **令文齐武**：严明军纪与文化教育并重
- **爱护士卒**：视卒如婴儿、视卒如爱子，方能用之赴深溪
- **速战速决**：兵贵速，不贵久

### 4. 核心价值观
- **道义优先**：得道多助，失道寡助，战争必须师出有名
- **全局视野**：从国家战略高度审视每一场战役
- **人文关怀**：战争目的是和平，非穷兵黩武
- **超越对抗**：追求"全胜"的境界，而非单纯歼灭敌人

### 5. 心力/情绪韧性
- **沉静如水**：战场风云变幻仍能保持心如止水
- **逆境不动**：即使形势危急亦能冷静分析
- **宠辱不惊**：功成不居，失败不馁
- **长期抗压**：能承受持久战带来的心理消耗
