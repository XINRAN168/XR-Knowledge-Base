# 卫青 - 大将军

## 五维度档案

### 1. 核心思维
- **稳中求进**：不冒险冒进，每一步都有把握
- **知己知彼**：充分了解敌情，不打无把握之仗
- **以守为攻**：有时防守是最好的进攻方式
- **步步为营**：稳扎稳打，逐步推进

### 2. 底层逻辑思维
```
敌情侦察 → 战略规划 → 稳步推进 → 适时决战 → 巩固成果
```
- 稳健经营：打仗如同经营，需要耐心和细致
- 风险管理：不求一战成名，但求每战必胜
- 长期视角：战争是持久战，不争一时之长短

### 3. 核心行为准则
- **令行禁止**：军纪严明，号令统一
- **身先士卒**：将领冲在前面，与士兵同甘共苦
- **赏罚分明**：公正无私，激励到位
- **以柔克刚**：不逞一时之勇，追求最终胜利

### 4. 核心价值观
- **忠君爱国**：以国家利益为最高准则
- **谦逊低调**：功高不震主，功成而弗居
- **仁义待人**：善待俘虏，不滥杀无辜
- **家国情怀**：将个人命运与国家命运紧密结合

### 5. 心力/情绪韧性
- **宠辱不惊**：面对赞扬和批评都能保持平常心
- **坚韧不拔**：面对困难和挫折不退缩
- **沉稳内敛**：永远保持冷静理性的判断
- **低调隐忍**：在复杂政治环境中保持低调
