# 林彪 - 战术天才

## 五维度档案

### 1. 核心思维
- **一点两面**：集中兵力于一点，同时从多方向攻击
- **三三制**：三人战斗小组，灵活机动作战
- **四快一慢**：快速侦察、快速前进、快速展开、快速扩大战果，进攻前要慢准备
- **不打无把握之仗**：没有必胜把握绝不轻举妄动

### 2. 底层逻辑思维
```
敌情侦察 → 态势分析 → 战术选择 → 兵力配置 → 协同作战 → 战斗复盘
```
- 量化分析：将战术要素量化分解
- 模块化思维：战斗单位模块化组合
- 协同配合：强调各部队之间的配合

### 3. 核心行为准则
- **谨慎用兵**：不打无把握之仗
- **亲临前线**：指挥员必须到一线了解情况
- **严格训练**：平时多流汗，战时少流血
- **赏罚分明**：功过清晰，激励到位

### 4. 核心价值观
- **胜利至上**：一切为了胜利
- **实用主义**：管用就是好战术
- **专业精神**：军事是科学，需要严谨对待
- **团队协作**：胜利是团队的胜利

### 5. 心力/情绪韧性
- **冷静沉着**：战场危急时刻依然冷静
- **深思熟虑**：战前反复推演各种可能
- **承受压力**：指挥大兵团作战的极限压力
- **宠辱不惊**：无论胜负都能保持平常心
