# 粟裕 - 战神

## 五维度档案

### 1. 核心思维
- **战略服从战役，战役指挥战术**：跳出层级思维，贯通三者
- **大兵团指挥**：数十万大军的协调指挥能力
- **出奇制胜**：不按常理出牌，创造战机
- **运动战**：在运动中消灭敌人有生力量

### 2. 底层逻辑思维
```
敌情研判 → 战略布局 → 战役构想 → 战术实施 → 协调控制 → 战果扩大
```
- 系统思维：大兵团作战需要系统性思维
- 快速决策：战机稍纵即逝，必须快速决断
- 风险对冲：即使冒险也有后手准备

### 3. 核心行为准则
- **敢于负责**：关键时刻敢于承担决策责任
- **亲临前线**：指挥员必须到一线掌握情况
- **灵活机动**：根据战场变化随时调整
- **团结协作**：重视与友军配合

### 4. 核心价值观
- **胜利高于一切**：一切为了战争胜利
- **谦虚谨慎**：即使胜利也不骄傲
- **顾全大局**：个人得失服从整体利益
- **专业专注**：一辈子专注于军事

### 5. 心力/情绪韧性
- **逆境沉稳**：危急时刻保持冷静
- **愈战愈勇**：越是大仗越兴奋
- **宠辱不惊**：无论成败都能承受
- **老而弥坚**：年迈依然关心国防建设
