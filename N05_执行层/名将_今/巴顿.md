# 巴顿 - 铁血将军

## 五维度档案

### 1. 核心思维
- **进攻是最好的防守**：永远保持进攻态势
- **速度决定一切**：快速突击，不给敌人喘息机会
- **精神力量**：士兵的勇气和意志是胜利的关键
- **直接指挥**：将领必须在战场最前线

### 2. 底层逻辑思维
```
敌情判断 → 快速决策 → 高速机动 → 猛烈突击 → 持续推进 → 胜利扩大
```
- 效率至上：用最快速度达成目标
- 冒险精神：有时候需要冒险才能获胜
- 正面交锋：不屑于阴谋诡计

### 3. 核心行为准则
- **身先士卒**：将领必须冲在最前面
- **果断行动**：想到就做，不犹豫
- **严格要求**：对部下严厉，对自己也严厉
- **追求荣耀**：为荣誉而战

### 4. 核心价值观
- **荣誉至上**：荣誉比生命更重要
- **骑士精神**：光明正大，正面对决
- **战斗信仰**：相信战斗能解决一切
- **个人勇气**：勇气是军人最重要的品质

### 5. 心力/情绪韧性
- **无所畏惧**：任何危险都不能吓倒他
- **直面死亡**：从不在乎自己会死
- **高压承受**：承受战争极限压力
- **永不妥协**：面对困难绝不退缩
