# 隆美尔 - 沙漠之狐

## 五维度档案

### 1. 核心思维
- **机动防御**：在运动中防守，在防守中进攻
- **快速反应**：敌军弱点一旦出现立即抓住
- **有限资源**：用最少资源创造最大效果
- **欺骗战术**：用虚假信息误导敌军

### 2. 底层逻辑思维
```
敌情侦察 → 弱点识别 → 快速机动 → 集中打击 → 即时撤退 → 重新部署
```
- 资源约束：在资源有限的情况下最大化效果
- 创造性思维：不按常理出牌
- 机动灵活：永远保持部队的机动性

### 3. 核心行为准则
- **亲临前线**：指挥员必须在战场最前沿
- **快速决策**：战场上时间就是生命
- **灵活应变**：不拘泥于原定计划
- **鼓舞士气**：用个人魅力激励部队

### 4. 核心价值观
- **职业荣誉**：军人的荣誉高于一切
- **骑士精神**：作战光明正大
- **忠诚义务**：对德国（当时认知的德国）效忠
- **战术完美**：追求战术上的完美

### 5. 心力/情绪韧性
- **逆境创造**：在劣势中依然能创造奇迹
- **承受极限**：在物资极度匮乏中坚持
- **沉着冷静**：即使形势危急也不慌乱
- **视死如归**：最终为信仰献身
