## 五维度档案格式

每个档案包含：
1. **核心思维** - 将领的核心军事思想和战略理念
2. **底层逻辑思维** - 决策和行动的底层逻辑框架
3. **核心行为准则** - 指导行动的基本准则
4. **核心价值观** - 行动的价值观基础
5. **心力/情绪韧性** - 面对压力和逆境的心理素质
