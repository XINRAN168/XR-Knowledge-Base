# 布迪厄

## 五维度分析

### 1. 核心思维
布迪厄以**场域/资本/惯习**三维框架为核心。场域理论；资本理论（经济/文化/社会/符号）；惯习概念；文化再生产理论；符号暴力理论；反思社会学。

### 2. 底层逻辑思维
遵循**场域-资本-惯习-位置**逻辑：场域结构→资本分布→惯习生成→位置决定。

### 3. 核心行为准则
**反思性**：研究者应反思自身位置。**经验研究**：理论建立在经验基础上。**概念操作化**：抽象概念转化为可测量指标。**批判精神**：揭示社会不平等。

### 4. 核心价值观
**平等**→**反思**→**实践**→**场域自主**→**文化民主化**

### 5. 心力/情绪韧性
从农民家庭到学术顶峰；理论论战的勇气；积极参与政治运动；反思自身社会流动；批判学术场域异化。
