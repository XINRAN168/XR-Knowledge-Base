# 韦伯

## 五维度分析

### 1. 核心思维
韦伯以**理想类型**和**理性化理论**为核心。理想类型方法论；合理性化理论；科层制理论；新教伦理与资本主义精神；支配类型学；价值无涉方法论原则。

### 2. 底层逻辑思维
遵循**类型-比较-理解**逻辑：概念工具→比较方法→理解社会学→多因素分析。

### 3. 核心行为准则
**学术诚实**：区分事实与价值判断。**概念精确**：创造和使用精确概念。**比较视野**：在比较中理解特殊性。**学术自由**：不受政治干预。

### 4. 核心价值观
**理性**→**自由**→**理解**→**方法论**→**责任伦理**

### 5. 心力/情绪韧性
与精神疾病斗争中坚持学术工作；一战期间积极讨论政治；"一个人如果不能期待这个世界在五十年后对他深感兴趣"的学术信仰。
